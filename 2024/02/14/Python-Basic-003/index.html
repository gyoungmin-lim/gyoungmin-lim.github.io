(Python 기초) 조건문, 반복문

Created2024-02-14|Updated2024-02-17|StudyPython
|Post Views:

조건식

조건식이란?

  • 어떤 조건에 따라 실행이 분기되는 식
  • A if 조건식 else B : 조건식의 결과가 True이면 A 실행, 그렇지 않으면 B 실행

[Ex] 적설량 입력하고, 적설량 30mm 이상이면 대설 경보 발령하고, 그렇지 않으면 대설 경보를 해제하는 코드를 작성해보자.




제어문

조건문

  • 특정 조건에 따라 프로그램을 분기
  • 조건문 종류
코드 의미
if 단일조건
if ~ else 양자택일
if ~ elif 다자택일(다중조건)


if

  • 실행문은 들여쓰기 필수 (문법구조)
    들여쓰기 한 부분 : 코드블럭

  • 형태
    if 조건식:

    `실행문`


if~else

  • 양자택일 조건문
  • 조건식 결과에 따라 둘 중 하나가 실행


pass 키워드

  • 실행문이 없을 경우, Error 발생
  • 추후 코딩 할 때 사용


len

  • 문자의 길이를 측정
  • 공백도 ‘문자’로 인식

[Ex] 소숫점 첫 번째 자리에서 반올림하는 프로그램을 만들어보자


조건식의 두 가지 사용법

  1. 조건식 결과에 따른 실행만 하는 경우
  2. 조건식 결과를 변수에 할당하는 경우
  • 모든 조건식(삼항연산자)은 if ~ else문으로 변경 가능
  • 단, 모든 if ~ else문을 조건식(삼항연산자)으로 변경할 수 있는 것은 아니다.

[Ex] 비 올 확률을 입력하고, 비 올 확률이 55% 이상이면 ‘우산 챙기세요.’ 그렇지 않으면 ‘양산 챙기세요.’를 작성하는 코드를 만들자.


다자택일 조건문

if ~ elif

  • 여러가지 조건식 결과에 따라 실행문이 결정

[Ex] 키오스크에서 메뉴를 선택하면 영수증을 출력하는 프로그램을 만들어보자.


다자택일 사용 시 주의점

  • 조건식 순서가 중요
  • 조건 범위를 명시할 것


중첩 조건문

  • 조건문 안에 또 다른 조건문 존재
  • 일반적으로 3-4단계 이상 중첩 조건문 사용 X (논리적 오류 발생 빈번)

[Ex] 출퇴근 이동수단에 따라 세금감면 해주는 정책을 세우려고 한다. 다음 내용에 맞게 프로그램을 만들어 보자.

출퇴근 대상인가?

  • 출퇴근 대상자이면
    • 도보, 자전거 > 세금 감면 5%
    • 버스, 지하철 > 세금 감면 3%
    • 자가용 > 추가 세금 1%
  • 출퇴근 대상자가 아니면
    • 세금 변동 없음
  • datetime : 현재 시스템의 시간, 날짜를 가져오는 module
  • 조건문 빠져나오기
    • while 조건문 사용 중, 반복문을 빠져오기 위해 변수 하나를 논리형(True)으로 설정 후, 구문 안에 대입
  • 조건에 의한 반복은 while, 횟수에 의한 반복은 for 구문이 유리함




반복문

반복문이란?

  • 특정 실행을 반복하는 것
  • 반복문을 사용하면 프로그래밍이 간결하며, 유지 보수가 쉬움


반복문 사용 이유

  • 잦은 실수(논리적 오류) 발생
  • 비효율적


반복문의 종류

  • 횟수에 의한 반복
  • 조건에 의한 반복


for문

  • 횟수에 의한 반복 for문이 주로 사용

  • 정해놓은 반복 횟수에 따라 반복 실행

    • for문도 if문과 동일하게, ‘pass’키워드 사용 가능


for ~ in

  • 기본 형태

[Ex]

[Ex] 구구단


반복 범위 설정

range

  • 반복할 범위를 설정하는 가장 기본적인 함수

  • 기본 사용 방법

    • 단계 매개병수는 생략 가능
    • 생략 시 1씩 증가


While문

  • 조건에 의한 반복

    • 조건에 만족하면 반복 실행, 그렇지 않으면 반복 중단
    • 조건에 의한 반복은 while문이 주로 사용

  • 무한루프에 빠질 경우 프로그램에 악영향을 미칠 수 있으므로 주의!


기본 구문

[Ex] while문을 사용하여 구구단 7단을 출력해보자.


반복문 비교

for문 while문
특 징 - 횟수에 의한 반복 - 조건에 의한 반복
예 제 - 1부터 10까지의 합을 구하는 경우 - 1부터 시작해서 7의 배수의 합이 50이상인 최초의 정수 출력

[Ex] 자동차 바퀴가 한 번 구를 때마다 0.15mm씩 마모된다고 한다. 현재의 바퀴 두께가 30mm이고 최소 운행 가능 바퀴 두께가 20mm라고 할 때, 앞으로 구를 수 있는 횟수를 구해보자.


무한루프

무한 반복 실행

  • 무한루프 : 반복문을 빠져나올 수 없는 경우
    1. while문에서 조건식의 결과가 항상 참인 경우
    2. 조건문에 논리형 데이터를 사용한 경우

[Ex] 하루에 독감으로 병원에 내방하는 환자 수가 50명에서 100명 사이일 때, 누적 독감 환자 수가 최초 10,000명을 넘는 날짜를 구해보자.


반복문 제어

continue

  • 반복 실행 중 continue를 만나면 실행을 생략하고, 다음 반복 실행문으로 넘어간다.


else

  • else의 실행문은 반복문이 종료된 후 실행된다.

[Ex] 7의 배수 갯수 구하기


break

  • 실행을 중단하고 반복문을 중단함

[Ex] 새끼 강아지 체중이 2.2kg가 넘으면 이유식을 중단하려고 한다. 한번 이유식을 먹을 때 체중이 70g 증가한다고 할 때, 예상되는 이유식 날짜를 구하자.
(현재 체중은 800g)


중첩 반복문

  • 반복문 안에 또 다른 반복문 선언

[Ex] 구구단 전체를 출력하자.
