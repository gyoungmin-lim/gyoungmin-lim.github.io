(Hwp) Introduction

Created2024-02-14|Updated2024-02-18|ToolOA
|Post Views:

Introduction

한글(Hwp)는 한컴(한글과 컴퓨터)에서 배포하는 프로그램으로, 국내 공적인 서류를 제출하는데 필수 불가결한 오피스이다. 학생신분일 뿐만 아니라, 국내에서 직장을 구하게 된다면 어디에서든 써먹을 수 있으니 기본정도는 해야한다고 생각한다. 개인적으로 한글을 사용하면서 느낀점에 대해 적어본다.

✅ MS Office(Microsoft Office)를 사용할 줄 안다면, Hwp에 그렇게 시간을 투자하지 말자

  • 개인적으로, 한컴오피스에서 제공하는 것….. 많이 불편하다. 너무 불편하다. 써먹지 못할 정도이다. 실제로 기업에서 MS Powerpoint를 쓰고, MS Excel을 쓰지, 한쇼, 한셀을 쓰는 곳을 봤는가?
  • 한컴오피스에서 써먹을 것은 한글(Hwp)밖에 없다. 참고하자.

✅ Hwp에 투자할 시간으로 Coding에 더 할애하자.

  • 말 그대로다. Coding은 연구에 간접적으로 연관이라도 되어있다.(시각화, ML, DL 등). 하지만 Hwp는 보고서 쓰려고 사용하는거지. 그마저도 필수적인 요소는 아니다. 요즘 Hwp 입지 많이 줄어들고 있다.

✅ 다른 프로그램을 사용해도 괜찮다.

  • 본인은 한글 경력만 20년이기 때문에 익숙해서 사용하는 것이다. 만약 지금 아무것도 몰랐었다면, 지금처럼 Notion을 쓰거나, Markdown 기반 프로그램(Typora, StackEdit 등)을 썼을 것이다. 정말 보고서 작성이나 공문서 작성용이 아니라 개인 공부목적으로 요약, 정리를 할 생각이라면 강추! 는 하지않는다. (시대가 Markdown같이 간단한 프로세스로 보편화되고 있기 때문)

단축키

본인은 보통 요약, 정리 하거나 스피디하게 회의록을 작성할 때, 보고서를 작성하는데 Hwp를 사용한다. 아래 단축키는 개인적으로 가장 많이 사용한다고 생각한 단축키들이다. (이외에도 엄청 많을 것인데, 다는 모름)


자주 사용하는 글꼴

본인이 한글에서 자주 사용하는 폰트를 정리해 봄(개인의 성향에 따라 원하는 글꼴을 사용하면 됨)

  • 폰트의 기본 설정은 함초롬바탕으로 설정되어있다. 이를 그대로 써도 무방하나, 네모난 타입 글씨를 좋아해서 맑은 고딕을 주로 사용한다. 이 외에도 아래 그림에 자주 사용했던 추천 글씨체들을 구분해놓았으니 참고. (’오이’체 은근 큐리하다. 가시성도 나쁘지 않음)
  • 제목의 대부분은 HY헤드라인M 과 HY견고딕을 사용함. 최근에는 대제목을 HY헤드라인M으로 설정하며, 소제목을 HY견고딕 또는 맑은고딕 + 진하게 를 사용한다.

수식

수학의 기본이면서, 수학에 빠질 수 없는 수식 파트이다. LaTeX 수식을 다뤄보았던 사람들이라면 쉽게 적응할 것이라 생각한다. 본인 직접 써본 결과 한글의 수식입력방법은 쓰레기다.

작업 표시줄에서는 [입력] - [수식 입력기] 의 방법으로 들어가며, 단축키로는 Ctrl + N + M으로 입력한다.

위, 아래 두가지로 나타나는데, 각각의 장점이 있다.

  • 위 : 알아서 칸을 만들어준다. 예를들면, 아랫첨자, 윗첨자를 쓸때 주로 중괄호 { } 를 사용하는데, 위에서 작성할 경우 만들어준다.
  • 아래 : 본인이 원하는 방향으로 직접 작성할 수 있다.

둘 다 사용해봤을때, 본인은 아래가 더 편함을 느꼈다.

Tips
수식을 작성하고 보통 괄호문제로 잘못 기입되는 경우가 있는데, 본인은 다음과 같이 수식을 작성할 때 진행한다.

  1. 아랫쪽에 수식 작성
  2. 윗쪽 클릭
  3. 알아서 괄호 설정 주루루루룩

LaTeX를 다뤄본 분들께서는 작성하는데 비슷하여 큰 어려움이 없을 것이다. 대부분의 수학기호, 그리스문자들의 표기가 비슷하다.
