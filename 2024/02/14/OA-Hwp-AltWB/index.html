(Hwp) 쪽 테두리 설정

Created2024-02-14|Updated2024-02-18|ToolOA
|Post Views:

단축키

  • Alt → W → B
    ※ 버전에 따라 다르니, 위치를 확실히 파악할 것


쪽 테두리/배경

  • 메뉴 표시줄의 [쪽] - [쪽 테두리/배경] 를 통해 설정


테두리/배경 종류

  • 양 쪽 : 홀수 + 짝수 페이지 전체 테두리/배경 적용
  • 홀수 쪽 : 홀수 페이지만 적용
  • 짝수 쪽 : 짝수 페이지만 적용


테두리

  • 페이지에 적용시킬 테두리 디자인 설정


위치

  • 종이 기준 : 용지 여백과 상관없이, 용지 크기를 기준으로 설정한 mm만큼 여백을 두고 테두리 설정
  • 쪽 기준 : 용지 여백을 기준으로 작성할 수 있는 페이지 내에서 설정한 mm만큼 여백을 두고 테두리 설정


적용 쪽

  • 위에서 설정한 테두리를 적용할 쪽 설정
