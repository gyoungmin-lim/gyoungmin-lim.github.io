Color Space

  • 색 공간은 색상을 전달할 수 있는 방법을 제공
  • 색 공간에서 모든 색상을 표현 가능


CIE LAB

국제 조명 위원회 (Commission internationale de l’éclairage)에서 1976년 를 이용한 Color Space를 정의

  • CIE XYZ 색 공간을 비선형 변환하여 만들어진 색 공간
  • Hunter 1948 Lab 색 공간을 지칭 ➡ 현대에 들어 CIE 1976 색 공간을 지칭
  • Lab 색 공간의 가장 큰 장점은 RGB나 CMYK와 달리 매체에 독립적
    • RGB, CMYK : 디스플레이 장비나 인쇄 매체에 따라 색이 달라짐
    • : 인간의 시각에 대한 연구를 바탕으로 정의

Notation

  • $L^*$ : 밝기(lightness)
  • : chromaticity coordinates


Notation

  • $L^*$ : 밝기(lightness)

    • ➡ 검은색, ➡ 흰색

  • $a^*$ : 빨강, 초록 중 어느쪽으로 치우쳤는지 나타냄

    • : 초록, : 빨강/보라

  • $b^*$ : 노랑과 파랑을 나타냄

    • : 파랑, : 노랑


Color Difference $\Delta E$ Formula

  • 성분의 color parameter를 특성화하기위해 사용
  • CIELAB $\Delta E$는 color space에서 색상 차이의 정도를 나타냄
  • 색상 차이의 방향(direction)은 설명되지 않음

Notation

  • $\Delta$ : 구체적인 color parameter의 차이를 나타냄
  • 이후 CIELAB 공식보다 더 나은 상관관계를 보인 색차 공식이 제시되었으며, 작은 색차에 대해 색 공간 균일성이 향상


CIELCH

  • CIELAB은 색 공간에서 계산을 할 때 Cartesian coordinates를 사용하나, CIELCH는 Polar coordinates를 사용
  • 증가 ➡ Chroma of color 또한 증가


Notation

  • $C^*$ : chroma
  • $h^*$ : hue angle
