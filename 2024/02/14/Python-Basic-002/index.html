(Python 기초) 데이터 입/출력, 산술 연산자(Operator)

Created2024-02-14|Updated2024-02-17|StudyPython
|Post Views:

데이터 입력

input 함수

  • input() 함수를 이용해서 입력한 데이터는 항상 문자(열) 자료형


형 변환 함수를 이용한 형 변환

[Ex] 삼각형 넓이

  • 사용자가 입력한 데이터를 이용, 도형 넓이 구하기

[Ex] 입력한 문자의 Type 파악하기




데이터 출력

Print 함수


콤마(,)를 이용한 데이터 연속 출력

  • 연속으로 출력하기 위해 사용
  • print 함수는 자동개행 옵션이 들어감
    • 자동개행 방지 방법
      • 빈 문자를 통해 막을 수 있다.


포맷 문자열을 이용한 데이터 출력


특수문자

  • \n : 개행
  • \t : 탭(Tab)

[Ex] 삼각형 넓이 구하기




Format 함수

Format() 함수를 이용한 출력


형식 문자를 이용한 데이터 출력

%s : 문자열
%d : 정수형
%f : 실수형


소숫점 자리수 정하기

  • 문자형 앞에 “.n” 입력 (n: natural number)




연산자

  • 연산자란?

    • 피연산자를 이용하여 계산
      Result = data1 + data2
      data1, data2 : 피연산자

  • 연산자 종류

연산자 종류
산술 연산자 +, -, , /, %, *, //
할당 연산자 =, +=, -=, *=, /=, %=, //=
비교 연산자 >, >=, <, <=, ==, !=
논리 연산자 and, or, not


산술 연산자

덧셈

  • 숫자를 이용한 계산
  • 문자를 이용한 계산
  • 정수와 문자는 계산할 수 없다.

[Ex] 국어, 영어, 수학 성적의 합계를 구하여라.


뺄셈


곱셈

  • 숫자(정수, 실수)를 이용한 곱셈
  • 문자(열)을 이용한 곱셈


나눗셈

  • 0으로 나눌 수 없다 (error 발생)

  • 나눗셈 결과는 항상 float형 이다.

[Ex] 국어, 영어, 수학 과목 합계와 평균 구하기


나머지

  • 나눗셈 결과 나머지 만 구함


몫

  • 나눗셈 결과 몫만 구함


divmod 함수

  • 몫과 나머지를 동시에 추출하는 함수

  • divmod(A, B) : A를 B로 나누었을 때 몫과 나머지
    (a, b) : 몫은 a, 나머지는 b)

[Ex] 몫과 나머지 구하기

[Ex] 한 모둠 별 학생 수 구하기


거듭제곱

  • 같은 수를 여러 번 곱한 값을 구함
  • n의 m제곱근 공식


sqrt 함수

  • 제곱근을 구함


pow 함수

  • 거듭제곱을 구함

[Ex] 아들이 엄마한테 용돈을 받는데, 첫 달에는 200원을 받고, 매월 이전 달의 2배씩 인상하기로 했다. 12개월 째 되는 날에는 얼마를 받을 수 있는지 계산해보자.


복합 연산자

  • 할당(대입)연산자
    • 오른쪽의 값을 왼쪽에 할당(대입)
  • 복합 연산자
    • 연산 후 할당하는 것을 간략하게 표현

[Ex] 단순 계산

[Ex] 연간 누적 강수량을 출력해보자


비교 연산자

숫자 비교

  • 연산 결과는 bool (True / False)

[Ex] 숫자 두 개를 입력한 후 비교 연산 결과를 출력하는 코드 작성해보자.

[Ex] 자동차 전장과 전폭을 입력하면 자동차 기계 세차 가능여부를 출력하는 코드를 작성해보자. (전장: 5200mm ,전폭: 1985mm)


문자 비교

  • 아스키 코드(ASCII Code)를 이용한 비교연산


Order 함수

  • 문자를 십진수(정수)로 표현


Chr 함수

  • 십진수를 문자로 표현

[Ex] 알파벳을 입력하면 아스키 코드를 출력하는 코드를 작성하자.

[Ex] 아스키 코드를 입력하면 문자를 출력하는 코드를 작성하자.


논리연산자

논리연산자 종류

  • and, or, not
A and B : A와 B 모두 True이면 True
A or B : A와 B 둘 중 하나만 True여도 True
not A : A의 상태를 부정하는 결과

[Ex] 연산자 비교하기

[Ex] 백신 접종 대상자 확인

[Ex] 과락여부 확인




Operator 모듈

모듈

  • 누군가 이미 만들어 놓은 기능
    • 수학 연산 관련 모듈
    • 난수 관련 모듈
    • 연산자 관련 모듈


산술 연산자 관련 함수

연산자 함수
+ operator.add()
- operator.sub()
* operator.mul()
/ operator.truediv()
% operator.mod()
// operator.floordiv()
** operator. pow()

[Ex] random과 operator 모듈을 사용해서 10 부터 100 사이의 난수 중 십의 자리와 일의 자리가 각각 3의 배수인지 판단하는 코드를 작성해보자.
