HYCOM 수온, 염도 데이터 분석 및 비교

Created2024-02-14|Updated2024-02-14|Project
|Post Views:

Summary

  • Goal : 전 지구 해양수치모델과 실제 관측자료(Temperature)의 수온과 염도(Salinity) 비교
  • 수행기간 : 2021. 01 ~ 2021. 04
  • 수행내용
    • 전 지구 해양수치모델 중 하나인 HYCOM을 통해 국내 적용가능성 확인
    • 모델 관측 값과 국내 기상청에서 수집하는 관측자료 중 대해구좌표에 따른 수온(Temperature)과 염도(Salinity)를 비교
    • 깊이에 따라 수온과 염도의 차이가 다르게 형성됨. 10m에서는 0.2℃의 온도차와 0.2psu의 염도차가 났으며, 주로 1~4월에서 차이를 보임. 50m의 경우 0.4℃로 큰 차이를 보임. 반대로 염도의 경우 0.1psu의 차이를 보임.

HYCOM

  • 미국에서 사용하는 해양수치모델
  • 위성으로부터 제공되는 해수면의 시공간적 변화를 포함하여 여러가지 관측자료를 모델에 동화시키기 때문에 다양한 해양 연구에 활용할 수 있음
  • 전 지구 규모의 수치모델이기 때문에 우리나라 근해의 연구에 활용되기 위해서는 모델결과의 정확성에 대한 검증이 필요함


Data Configuration

  • HYCOM Data

    • 파일형태 : 200001010131.nc4
    • 2000년 01월 ~ 2010년 12월 데이터
    • 데이터 구성 : 월별(1개월) 데이터로 이루어져 있음
    • Variable
      Lon, lat, depth, time, salinity, water_temp, water_u, water_v
  • 대해구격자 rho_point
    • 위상해양정보시스템 (Satellite ocean information system)
    • csv 파일


Interpolation

  • 대해구격자 좌표의 Temperature, Salinity를 구하기 위한 Data Interpolation

How to?

  1. 대해구 좌표에 따른 Boundary를 설정
  2. Boundary 내 Data를 Interpolation하여 대해구 좌표의 Temperature, Salinity 값을 구함
  • Grid 설정
    • 대해구격자 rho_point 좌표를 기준으로 $\pm 0.25^\circ$의 격자 생성
Generate Grid with rho_point Set the Grid on observed data


Interpolation Method - Inverse Distance Weighting(IDW)

  • 인접한 점들의 값 $(x_i, y_i, z_i)$들을 사용하여 찾아내고자 하는 위치의 값 $(x_p, y_p, z_p)$를 거리의 역함수로 찾아내는 기법
  • $(x_i, y_i, z_i)$ 값을 알 때, $(x_p, y_p)$ 위치의 $z_p$ 값을 계산할 수 있음
  • $d$ : Distance
  • 두 점 사이 $(x_1, y_1)$, $(x_2, y_2)$의 거리


Case : One Data Set Test

  • 2006년 01월 01일 ~ 2006년 01월 31일 데이터 이용
    • HYCOM data information: lon, lat, depth, temperature
  • 01월 01일의 데이터 정제를 목표
  • 위도, 경도에 따른 깊이 10m의 수온 데이터를 정제
  • 2006년 01월 01일 Depth 10m의 수온을 Figure out

Try : near 16 points

  • IDW 알고리즘을 사용하는데 있어 구하고자 하는 Point와 인접한 16개의 점을 이용하여 수온 값을 Interpolation
  • rho_point 좌표 lon = 119.75, lat = 42.75 지점에서 Error 발생
    • Data의 개수가 16개 미만(6개) 인 좌표가 있어 IDW 알고리즘 적용 불가.


Use all data in boundary

  • Boundary 내부에 있는 모든 점을 이용하여 Interpolation 진행
    ※ 단, 수온 값이 NaN인 point는 제외하고 IDW Algorithm 진행

  • green dots represent using data and purple dots represent vanishing data

near 16 points Use all data

Conclusion

  • Depth = 10m

    • Temperature
      기존 값과 약 0.2℃의 차이를 보이며, 대체적으로 1~4월 Data에서 차이가 발생

    • Salinity
      약 0.25psu의 차이를 보이며, Interpolation결과가 전체적으로 낮게 형성

    • The temperature(left) and salinity(right) difference between Origin and HYCOM observations in depth 10m

  • Depth = 50m

    • Temperature
      기존 값과 약 0.4℃의 차이를 보이며, 전반적으로 Interpolation 결과가 낮게 나왔으나, 7~9월은 비슷하거나 높게 형성됨

    • Salinity
      약 0.1psu의 차이를 보임
