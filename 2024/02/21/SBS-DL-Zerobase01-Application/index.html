전체 Dataset에 Model 적용에 앞서, One DataSet을 우선 적용

Case : One Dataset on Index 17

  • 모델 구동을 위한 Module Load
  • 전체 데이터셋 Load
  • Target Point 지정
  • Index 17 (머리 드라이기로 말리기) Group 1번 Camera 001를 One Dataset으로 설정
  • Min-Max Scaler 적용하여 Train/Test set 분할
    • Train : Test = 8 : 2
  • Sequence 생성

Set Index

  • Index로 ‘설정했을 때‘와 ‘설정하지 않았을 때‘의 Sequence 생성 방법이 다름
    • 현 프로젝트를 진행하면서 세팅하는 Index는 Frame으로, 시간(time)을 나타냄
  • 이유는 아직 찾지 못했으나, 만일의 경우를 대비하여 두 개의 function으로 만들어 놓음
  • 두 Case 중 택 1
  • Case 1: Set Index 설정했을 때
  • Case 2: Set Index 설정 안했을 떄
  • Sequence 길이 설정
  • LSTM 모델 적용
