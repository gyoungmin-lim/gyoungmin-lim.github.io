(Statistics) 통계학 기초 - 추정

Created2024-02-21|Updated2024-03-14|StudyMathematics
|Post Views:

추정

  • 모집단의 모수를 모를 경우 표본으로 추출된 통계량을 모집단의 근사값으로 사용하는 것
  • 추정량(Estimator) : 표본평균으로 모평균을 추정할 때 표본 평균


추정방법

  • 점 추정(point estimation)

    • 모수를 하나의 특정값으로 추정하는 방법

  • 구간 추정(Interval estimation)

    • 모수가 포함될 수 있는 구간을 추정하는 방법


점 추정(Point Estimation) 성질

  • 일치성(Consistency)

    • 표본의 크기가 모집단의 크기에 근접해야 함
    • 표본의 크기가 클수록(모집단에 가까울수록) 추정량의 오차가 작아짐

  • 불편성(Unbiased Estimator)

    • 추정량이 모수와 같아야 함
    • 모수가 $\theta$, 추정량이 $\hat{\theta}$라고 가정
      $E(\hat{\theta}) = \theta$ 일 경우, 이 추정량을 불편 추정량이라고 함
      추정량이 같지 않을 경우, 편의(biased)라고 함

  • 유효성(Efficiency) : 추정량의 분산이 최소값이어야 함

    • 모수에 대한 추정량의 분산이 작을수록 추정량이 효율적
    • 모수 $\theta$에 대하여 불편 추정량이 $\hat{\theta_1}$, $\hat{\theta_2}$이라고 가정
      $Var(\hat{\theta_1}) < Var(\hat{\theta_2})$일 경우, $\hat{\theta_1}$이 효율적인 추정량

  • 평균오차제곱(Mean Squared Error, MSE)

    • 평균오차제곱이 최솟값이어야 함
    • 즉, $E \left( (\hat{\theta} - \theta)^2 \right)$이 최소여야 함


구간추정

  • 모수가 포함될 수 있는 구간을 추정하는 방법

신뢰구간(Confidence Level)

  • 추정값이 존재하는 구간에 모수가 포함될 확률
  • 신뢰 수준 : $100 * (1-\alpha)$% 이 때, $\alpha$ : 오차수준

[Ex] 신뢰 수준 95% $\iff$ 구간 추정값 오차가 발생할 확률 : 5%

$\iff p=0.05$ : 유의 수준(Signifiant Level)

  • 신뢰 구간은 신뢰 하한, 신뢰 상한으로 표시
    • 추정 모수 : $\theta$

[Ex] 표본 평균이 $\bar{x}$이고, 표준오차가 $sd$라고 할 때 모평균 $\mu$를 추정하는 신뢰구간

  • 신뢰구간의 의미
    • $P(-z \leq Z \leq z) = 0.95$일 때, $z = 1.96$

[Ex] 표본 평균이 50, 표본표준편차가 10이라고 할 때, 신뢰구간 95%에서 평균 추정

sol)


모집단의 분산을 아는 경우

  • $X_1, X_2, \cdots, X_n \sim N(\mu, \sigma^2), \qquad \hat{\mu} = \bar{X} \sim N \left( \mu, \frac{\sigma^2}{n} \right), \qquad Z \sim N(0, 1)$

  • $\mu$의 95% 신뢰구간

    $
    \iff P \left( -1.96 \leq \frac{\bar{x}-\mu}{\sigma/ \sqrt{n}} \leq 1.96 \right) = 0.95 \
    $
    $
    \iff P \left( -1.96 \cdot \frac{\sigma}{\sqrt{n}} \leq
    \bar{x} - \mu
    \leq
    1.96 \cdot \frac{\sigma}{\sqrt{n}} \right) = 0.95
    $

    $\iff P \left( \bar{x} - 1.96 \cdot \frac{\sigma}{\sqrt{n}} \leq \mu \leq \bar{x} + 1.96 \cdot \frac{\sigma}{\sqrt{n}} \right) = 0.95$

[Ex] 모집단의 분산이 4, 표본평균이 10, 표본의 크기가 25일 때, 평균의 95% 신뢰구간

sol)

$\sigma^2 = 4, \bar{X} = 10, n = 25$


모집단의 분산을 모르는 경우

  • $X_1, X_2, \cdots, X_n \sim iid \ N(\mu, \sigma^2), \qquad \Gamma = \frac{\bar{X} - \mu}{s/\sqrt{n}} \sim t(n-1)$
  • $\mu$의 95% 신뢰구간

[Ex] 표본분산이 4이고, 표본 평균이 10, 표본의 크기가 25일 때, 평균의 95% 신뢰구간($t_{0.25, 24} = 0.685$)

sol)

$n = 25, \ s^2 = 4, \ \mu = 10$, 95% 신뢰구간이므로, $\alpha = 0.5$


표본의 크기 결정

허용오차(Permissible Error)

  • 추정한 값이 틀려도 허용할 수 있는 오차
  • 정규분포의 신뢰구간을 통해 허용 오차를 계산

$\iff d = z_{\alpha/2} \cdot \frac{\sigma}{\sqrt{n}}$

$\iff d \cdot \sqrt{n} = z_{\alpha / 2} \cdot \sigma$

$\iff \sqrt{n} = z_{\alpha/2} * \frac{\sigma}{d}$

$\iff n = \left( \frac{z_{\alpha/2} \cdot \sigma}{d} \right)^2$




모비율 추정

모비율의 점 추정

  • 비율에 대한 추정
  • 원하는 속성(Class)에 속할 경우 ‘1’, 아니면 ‘0’일 때, 1의 속성을 갖는 것의 갯수를 $X$라고 하면 $X \sim B(n, \ p)$
  • 이 때, 모비율의 점 추정량을 표본 비율(Sample Proportion)이라 함 $(\hat{p} = \frac{X}{n})$
