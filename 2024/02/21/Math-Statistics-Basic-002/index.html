(Statistics) 통계학 기초 - 모집단(Population), 표본(Sample), 표본 분포(Sampling Distribution)

모집단 (Population)과 표본 (Sample)


표본추출 (Sampling)

  • 모집단으로부터 표본을 추출하는 것
  • 표본으로부터 그 특성을 찾아내고 모집단의 특성을 추론하고자 함


복원추출 (Sampling with replacement)

  • 모집단에서 데이터를 추출할 때 하나를 추출하고 다시 넣고 추출하는 방법
  • 동일한 표본이 추출될 수 있음


비복원추출 (Sampling without replacement)

  • 모집단에서 데이터를 추출할 때 하나를 추출하고 다시 넣지 않고 추출하는 방법


Random Sampling

  • 각 개체가 모두 동일한 확률로 추출하는 방법
  • 모집단에서 데이터를 추출할 때 편향되지 않도록 주의해야 함


불균형 데이터 (Imbalanced Data)의 문제

  • 데이터가 불균형 데이터일 경우 문제 발생

예측모형 개발 목적

  • 대상이 발생할 확률 예측하는 경우가 대부분
  • 예측 대상이 전체 대비 낮을 경우, 모델 성능이 좋지 않을 수 있음
  • [Ex] 신용평가 모델, 제조 불량 예측 등

해결방안

  1. Sampling 기법을 통해 해결
  2. 모델을 통한 성능 개선
    [Ex] Cost-Sensitive Learning


Sampling 기법

  • 타겟 대상의 비율이 낮은 경우

  • Over Sampling

    • 적은 Target Data의 수를 많은 Class의 비율만큼 증가시키는 기법 (일정 비율로 복원 추출 하는 개념)
    • 과적합(OverFitting)의 문제 발생 가능성

  • Under Sampling

    • 많은 Target Data의 수를 적은 Class의 비율만큼 감소시키는 기법
    • 임의로 추출한 데이터가 biased(편향)될 수 있고, 모델의 성능이 떨어질 수 있음

통계량(Statistic)

  • 표본에 기초하여 계산되는 수치 함수

카이제곱 분포 (Chi-square Distribution)

  • 확률변수 $Z_1^2, Z_2^2, \cdots, Z_n^2$가 표준 정규분포를 따를 경우, 확률변수 $Z$에 대하여

자유도(degree of freedom)

  • 표본수 - 제약조건의 수 또는 표본수 - 추정해야하는 모수의 수를 의미
  • 일반적으로 $n-1$

1, 2, 3, 4를 선택했을 경우 나머지 수는 자동으로 5가 되어야 평균이 5로 정해짐

  • 카이제곱 분포는 자유도 $v$의 크기에 따라 모양이 달라짐
    • 자유도가 커질수록 분포가 좌우 대칭 형태
    • 자유도가 커질수록 표준정규 분포에 근사하며, $v \geq 30$이면 확률을 근사적으로 정규분포로 구할 수 있음

  • $Y_1 \sim \chi^2(v_1), \ Y_2 \sim \chi^2(v_2)$이면
