(Statistics) 통계학 기초 - 확률(Probability), 확률변수(Random Variable), 확률분포(Probability Distribution)

확률 (Probability)

  • 모든경우의 수에 대한 특정 사건이 발생하는 비율
  • 0과 1 사이의 확률에 100을 곱하여 0과 100 사이의 백분율(%)로 나타냄
  • 할·푼·리로 나타내기도 함


표본 공간 (Sample Space)

  • 표본 공간이란 어떤 실험에서 나올 수 있는 모든 가능한 결과들의 집합
  • 사건 $A$가 일어날 확률을 $P(A)$라고 하고, 표본 공간 $S$가 유한집합 일 때 표본 공간의 모든 원소들이 일어날 확률이 같을 경우

[Ex] 동전던지기

  • 앞면 : Head($H$), 뒷면 : Tail($T$)

  • 표본공간 $S = { H, T}$

  • 주사위던지기

    • 표본공간 $S = { 1,2,3,4,5,6 }$

[Ex] 동전 두 개를 던져서 뒷면이 한번만 나오는 확률

  • 앞면 : Head($H$), 뒷면 : Tail($T$)
  • 표본공간 $S = { (H, H), (H, T), (T, H), (T, T) }$
  • 사건 $A = { (H,T), (T,H) }$


통계적 확률

  • 상대도수 : 어떤 시행을 $N$번 반복했을 때, 사건 $A$에 해당하는 결과가 $r$번 일어난 경우
  • $N$이 무한히 커지면 상대도수는 일정한 수로 수렴

이를 사건 A의 통계적 확률 또는 경험적 확률이라고함

[Ex]

  • 타자가 타석에서 안타를 칠 확률
  • 공정에서 제품이 정상일 확률(수율)


확률의 성질

집합 성질

  • 합사건(Union)

    • 사건 $A$ 또는 $B$가 일어날 확률
    • $A \cup B$

  • 곱사건(Intersection)

    • 사건 $A$와 $B$가 동시에 일어날 확률
    • $A \cap B$

  • 배반사건(Mutually Exclusive Event)

    • 사건 $A$와 $B$가 동시에 일어날 수 없을 경우
    • 독립(Independent) 관계라고도 함.
    • $A \cap B = \emptyset$

  • 여사건(Complement)

    • 사건 $A$가 일어나지 않을 경우
    • $A^c$


확률 법칙

  • $P(A\cup B) = P(A) + P(B) - P(A\cap B)$
  • $P(A \cap B) = P(\emptyset) = 0$ $\quad \quad \text{If 사건 $A$와 $B$가 독립(Independent)일 경우}$
  • $A$의 여사건이 $A^c$이면, $P(A) + P(A^c) = 1$


순열 (Permutation), 조합 (Combination)

  • Factorial
    • $n$개를 일렬로 늘여 놓은 경우의 수
    • $n!$ 로 표기
  • 순열(Permutation)
    • 순서를 고려
    • $n$개 중 $r$개를 뽑아서 배열하는 경우의 수
  • 조합(Combination)
    • 순서를 고려하지 않음
    • $n$개 중 $r$개를 뽑아서 배열하는 경우의 수


조건부 확률 (Conditional Probability)

  • 어떤 사건 $A$가 발생한 상황에서 사건 $B$가 발생할 확률
  • 확률의 곱셈법칙

※ 사건 $A$와 $B$가 독립일 경우, $P(A \cap B) = P(A) \cdot P(B)$


베이즈 정리(Bayes’ Theorem)

  • 표본 공간 $S$에서 서로 독립인 사건 $B_1 , B_2 , \cdots, B_k$에 의해 분할되어 있을 때, 임의의 사건 $A$에 대해 다음이 성립




확률 변수 (Random Variable)

  • 표본공간에서 각 사건에 실수를 대응시키는 함수
  • 확률 변수의 값은 하나의 사건에 대하여 하나의 값을 가지며, 실험의 결과에 의하여 변함

  • 일반적으로 확률 변수는 대문자$(X, Y)$로 표현하며, 확률변수의 특정 값을 소문자$(x, y)$로 표현

  • 확률 변수의 평균

    • 기댓값(Expected Value)이라고도 함
  • 확률 변수의 분산


성질 (Property)

기대값 (Expected Value)의 성질

$a,b$가 상수이고, $X, Y$를 임의의 확률 변수라고 할 때, 다음이 성립

  • $E(a) = a$
  • $E(aX) = aE(x)$
  • $E(aX + b) = aE(x) + b$
  • $E(aX \pm bY) = aE(X) \pm bE(Y)$
  • $E(XY) = E(X)E(Y) \qquad if \quad X,Y \ \ is \ \ \ independent$


분산 (Variance)의 성질

$a,b$가 상수이고, $X, Y$를 임의의 확률 변수라고 할 때, 다음이 성립

  • $Var(a) = 0$
  • $Var(aX) = a^2 Var(X)$
  • $Var(X+Y) = Var(X) + Var(Y) + 2 \text{Cov}(X, Y)$
  • $Var(aX \pm bY ) = a^2 Var(X) \pm b^2 Var(Y) + 2\text{Cov}(X,Y)$
  • $Var(XY) = 0 \qquad if \quad X,Y \ \ is \ \ \ independent$
  • $Var(X) = E(X^2) - [E(X)]^2$


공분산 (Covariance)

  • 2개의 확률변수의 선형 관계를 나타내는 값
  • $\text{Cov}(X,Y) > 0$ : $X$가 상승할 때 $Y$도 상승 (양의 상관관계)
  • $\text{Cov}(X,Y) < 0$ : $X$가 상승할 때 $Y$는 하락 (음의 상관관계)
  • $\text{Cov}(X,Y) \approx 0$ : $X,Y$는 관계가 없음




확률 분포 (Probability Distribution)

  • 확률 변수 $X$가 취할 수 있는 모든 값과 그 값을 나타날 확률을 표현한 함수


이산 확률 분포 (Discrete Probability Distribution)

Summary


이산형 균등분포(Discrete Uniform Distribution)

  • 확률 변수 $X$가 유한개이고, 모든 확률 변수에 대하여 균일한 확률을 갖는 분포
  • 기댓값
  • 분산

[Ex] 주사위를 한 번 굴려서 나오는 숫자를 확률변수 $X$라고 하자.


베르누이 분포 (Bernuoulli Distribution)

베르누이 시행 (Bernuoulli Trial)

  • 각 시행의 결과가 성공, 실패 두가지 결과만 존재하는 시행
  • 베르누이 시행에서 성공이 1, 실패가 0의 값을 가질 때 확률 변수 $X$의 분포
  • 기댓값
  • 분산

[Ex] 파란 공 7개, 빨간 공 3개가 들어있는 주머니에서 공 하나를 뽑을 때, 파란 공이면 성공, 빨간 공이면 실패인 실험을 한다고 가정하자. 이 때 베르누이 분포를 정의하면


이항분포 (Binomial Distribution)

  • 연속적인 베르누이 시행을 거쳐 나타나는 확률 분포
  • 서로 독립인 베르누이 시행을 $n$번 반복해서 실행했을 때, 성공한 횟수 $X$의 확률 분포
  • 기댓값
  • 분산

[Ex] 축구선수의 패널티킥 성공률이 80%일 때, 10번의 기회에서 $x$번 성공횟수와 그 확률

[Ex] 반도체 공장에서 불량이 발생할 확률이 10%라고 하자. 10개의 제품을 생성했을 때 불량이 2개 이하일 확률을 구하시오.


포아송 분포 (Poisson distribution)

  • 어느 희귀한 사건이 어떤 일정한 시간대에 특정한 사건이 발생할 확률 분포

조건 (Condition)

  1. 어떤 단위구간(예,1일)동안 이를 더 짧은 작은 단위의 구간(예:1시간)으로 나눌 수 있고 이러한 더 짧은 단위구간중 어떤 사건이 발생할 확률은 전체 척도 중 항상 일정
  2. 두 개 이상의 사건이 동시 발생할 확률은 0에 가까움
  3. 어떤 단위구간의 사건 발생은 다른 단위구간의 발생으로부터 독립적
  4. 특정 구간에서 사건 발생확률은 그 구간의 크기에 비례
  5. 포아송 분포의 확률 변수 기댓값 $E(X)$와 분산 $Var(X)$ 모두 $\lambda$
    • 즉, 기댓값과 분산이 동일함

이항 분포의 포아송 근사

  • 확률변수 $X$가 $X \sim B(n, p)$이고, $n$이 충분히 크고 $p$가 아주 작을 때 $X$의 분포는 평균이 $\lambda = np$인 포아송 분포로 근사 시킬 수 있다.
    $n = 30$일 경우 크다고 판단
  • 보통 $n$이 클 때 $np<5$를 만족하는 $p$가 존재하면 근사 정도가 좋다고 함

[Ex] 이항분포와 포아송분포 비교

$n = 100, \ \ p = 0.01$ 이항 분포 $X \sim B (100, 0.01)$

$\lambda = 100 \times 0.01 = 1$ 포아송 근사 $X \sim \text{Poisson}(1)$

엑셀 함수

  • 이항분포 : BINOM.DIST($X,N,P,$ 누적여부)
  • 포아송분포 : POISSON.DIST($X, \lambda,$ 누적여부)
    ➣ 누적여부 = 1 (Yes), 누적여부 = 0 (NO)


기하 분포 (Geometric Distribution)

  • 어떤 실험에서 처음 성공이 발생하기까지 시도한 횟수 $X$의 분포
  • 각 시도는 베르누이 시행을 따름
  • 기댓값
  • 분산

[Ex] 축구선수 손흥민의 필드골 성공 확률이 30%일 때, 5번째 슛팅에서 골을 넣을 확률분포

$x = 5, \ \ p = 0.3$


음이항 분포(Negative Binomial Distribution)

  • 어떤 실험에서 성공확률이 $p$일 때, $r$번의 실패가 나올 때까지 발생한 성공 횟수 $X$의 확률분포
  • 기댓값
  • 분산

[Ex] 농구 선수 허훈의 자유투 성공 확률이 90%일 때, 3번째 실패가 나올 때까지 성공시킨 자유투가 10번일 확률

$x = 3 , \ \ r = 10$

확률밀도함수 (Probability Density Function, pdf)

    1. 모든 $X$에 대해서 $f(x) \geq 0$
    2. $P(x \in \left( -\infty, \infty) \right ) = \int_{-\infty}^{\infty} f(x) = 1$
    3. $P(a \leq X \leq b) = \int_{a}^{b} f(x) dx$

지수분포 (Exponential Distribution)

  • 단위 시간당 발생할 확률 $\lambda$인 어떤 사건의 횟수가 포아송 분포를 따를 때, 어떤 사건이 처음 발생할 때까지 걸린 시간 확률변수 $X$
  • 연속되는 사건 사이의 대기시간도 지수 분포

[Ex] 버스 정류장에서 버스가 100번 도착하는 횟수가 포아송 분포를 따른다고 가정하면, 첫 번째 버스가 도착할 때까지의 시간 분포가 지수분포
➣ 두 번째 버스가 도착하고 세 번째 버스가 도착 할 때까지의 대기시간의 분포도 지수분포

  • 지수분포의 확률밀도함수(pdf)

지수분포의 무기억성 (Memoryless Property)

  • 어떤 시점부터 소요되는 시간은 과거 시간에 영향을 받지 않음
  • Ex 1. 버스를 기다리는 대기시간은 먼저 기다린 사람의 확률과 같음
  • Ex 2. 전구를 한 달동안 사용했을 때 남은 수명은 한달 간 사용했던 영향을 받지 않음. 즉 새 전구와 한달 간 사용한 전구의 남은 수명은 같다고 생각함
  • 생존분석에서는 Weibull 분포 또는 long-normal 분포를 사용하여 예측
