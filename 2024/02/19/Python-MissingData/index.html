(Python) Missing Data(NA, NaN) 결측 값 처리

Created2024-02-19|Updated2024-02-19|StudyPython
|Post Views:

결측값 처리

  • Pandas를 이용하여 NA, NaN 데이터를 처리하는 기본적인 방법
  • 데이터 분석에 있어 가장 중요한 과정 중 하나
  • 결측값, 특이값을 얼마나 잘 처리하는지의 유무


결측값이나 특이값 처리 방법

  1. 무시한다
  2. 제거한다
  3. 다른 값으로 대체한다


Example

  • Pandas와 Numpy를 이용해 결측값(NaN)이 포함된 DataFrame 생성


무시한다

  • 말 그대로, 결측값을 무시하고 진행
  • 추후 데이터 분석을 하는 데 있어 다른 방법으로 잘 처리해야한다는 전제가 있음


dropna()

  • NaN이 하나라도 들어간 행을 제거하는 방법
  • how의 default 값이 ‘any’ 이다. 즉 두 코드는 같은 의미

dropna option : how

  • 기본값(Default)은 ‘any’
    • any : 하나라도 결측값이 존재한다면, 그 결측 값이 존재하는 Row 또는 Column을 삭제
    • all : 특정 Row 또는 Column의 모든 값이 NaN인 경우 삭제
  • 행, 열을 설정
    • axis = 0 (행)
    • axis = 1 (열)
how = ‘any’ how = ‘all’
NaN이 포함된 Row(Column) 전체 삭제 세 번째 Row(Cloumn)만 삭제


fillna()

  • NaN 값을 다른 값으로 대체하는 방법
  • 원하는 특정 값을 fillna()의 parameter 중 하나인 value에 설정해주면 NaN값이 변경
  • 열마다 다른 값을 입력하여 변경


fillna 함수의 기능 - limit

  • 연속된 NaN 값 중 $N$번째 값까지 변경을 원할 경우, limit을 통해 지정

‘A’, ‘C’, ‘D’ 열은 NaN값이 전부 채워졌지만, ‘B’ 열의 경우 두 번째 값까지 변경된 것을 확인할 수 있음


fillna 함수의 기능 - method

  • method = ffill 과 method = bfill 두 가지가 존재
ffill bfill
- forward fill
- 존재하는 데이터로부터 앞(=아래) 방향으로 채우라는 뜻		 - back fill
- 반대로 존재하는 데이터로부터 뒷(=위) 방향으로 채우라는 뜻
  • method = ‘ffill’
    • 윗 값으로 아랫방향으로 NaN값 대체
  • method = ‘bfill’
    • 아랫 값으로 윗방향으로 NaN값 대체
