(Python 중급) 자료구조 - 리스트(List)

Created2024-02-16|Updated2024-02-17|StudyPython
|Post Views:

자료구조

  • Python의 다양한 자료구조
  • 자료구조
    • 컨테이너 자료형 : 여러 개의 데이터가 묶여있는 자료형
    • 컨테이너 자료형의 데이터 구조를 자료구조라고 한다.
  • 각각의 컨테이너 자료형에 따라 차이가 있으며, 대표적인 컨테이너 자료형으로는 리스트(List), 튜플(Tuple), 딕셔너리(Dic), 세트(Set)가 있다.
    • 리스트(List)와 튜플(Tuple)의 차이점
      • 튜플(Tuple)은 수정이 불가하다.
    • 딕셔너리(Dictionary)는 키값과 키값에 해당하는 값(Value)으로 구성되어 있는 것.
    • 세트(Set) : 반복된 값이 존재하지 않음


리스트

  • 배열과 같이 여러 개의 데이터를 나열한 자료구조
  • '[]’ 를 이용해서 선언하고, 데이터 구분은 ‘,’를 이용
  • 숫자, 문자(열), 논리형 등 모든 기본 데이터를 같이 저장할 수 있음
  • 리스트에 또 다른 컨테이너 자료형 데이터를 저장할 수 있음.

[Ex] 가족 이름을 리스트에 저장하고 출력

[Ex] 오늘 일정을 리스트에 저장하고 출력


리스트 아이템 조회

  • 인덱스 : 아이템에 자동으로 부여되는 번호표
    • 0부터 순차적으로 저장
  • 리스트 아이템은 인덱스를 이용해서 조회 가능
  • 범위 밖의 인덱스 조회할 경우, 에러 발생
  • 리스트에서 아이템을 개별적으로 뽑았을 때, Type은 유지

[Ex] 5명의 학생 이름을 리스트에 저장하고, 인덱스가 홀수인 학생과 짝수인 학생을 구분해서 인덱스와 학생 이름을 출력해보자.


리스트 길이

  • 리스트의 아이템 개수를 확인

  • 리스트 길이 : 리스트에 저장된 아이템의 개수

  • len() : 문자열, 리스트, 목록, 튜플, 딕셔너리 등의 길이를 반환함

  • len()과 반복문을 이용하면 리스트의 아이템 조회가 가능
  • len() 함수는 리스트의 갯수 뿐만 아니라 문자열의 길이도 알 수 있다.

[Ex] 좋아하는 운동 종목을 리스트에 저장하고 반복문을 이용해 출력해보자.

  • 방법 1 : for문 사용
  • 방법 2 : while문 사용
  • len() 함수를 이용하지 않아도 리스트 내 아이템들을 출력할 수 있다.


리스트와 for문

  • for문을 이용하면 리스트의 아이템을 자동으로 참조할 수 있다.

[Ex] 차량 조회

  • 방법 1
  • 방법 2

리스트 내 또 다른 리스트의 아이템을 조회할 수도 있다.

  • 변수 1개 설정
  • 변수 $n$개 설정


[Ex] 아래 표와 리스트를 이용해서 학급별 학생 수와 전체 학생 수 그리고 평균 학생 수를 출력해보자

[Ex] </span> For문과 If문을 이용해서 과락 과목 출력하기

  • continue 사용

[Ex] 사용자가 국어, 영어, 수학, 과학, 국사 점수를 입력하면 과락 과목과 점수를 출력하는 프로그램을 만들어보자.


리스트와 While문

  • While문을 이용하면 다양한 방법으로 아이템 조회가 가능

[Ex] 아래 표와 리스트를 이용하여 학급별 학생 수와 전체 학생 수 그리고 평균 학생 수를 출력해보자.

[Ex] While문과 If문을 이용해서 과락 과목 출력

  • continue 사용

[Ex] while문을 이용해서 학급 학생 수가 가장 작은 학급과 가장 많은 학급을 출력해보자


enumerate()함수

  • 인덱스와 아이템을 한 번에 조회
  • 리스트를 사용할 때 유용
  • 문자열에도 사용 가능

[Ex] 가장 좋아하는 스포츠가 몇 번째에 있는지 출력하는 프로그램을 만들어보자.

[Ex] 사용자가 입력한 문자열에서 공백의 개수 출력


리스트에 아이템 추가

  • .append() : 마지막 인덱스에 아이템 추가

[Ex] 가족 구성원에 새로 태어난 동생을 추가해보자.


리스트 특정 위치 아이템 추가

  • .insert() : 특정 위치(인덱스)에 아이템을 추가

[Ex] 오름차순으로 정렬되어 있는 숫자들에 사용자가 입력한 정수를 추가하는 프로그램을 만들어보자. (단, 추가 후에도 오름차순 정렬이 유지되어야 한다.)


리스트 아이템 삭제

  • .pop() : 마지막 인덱스에 해당하는 아이템 삭제

[Ex] 점수표에서 최고, 최저점수를 제외하는 프로그램을 만들어보자.


리스트 특정 아이템 삭제

  • .remove() : 리스트 내 특정 아이템 삭제
  • remove()는 한 개의 아이템만 삭제 가능
    • 2개 이상의 데이터를 삭제 할 경우, while문을 이용
  • while ~ remove() 진행

[Ex] 사용자가 입력한 과목을 삭제하는 프로그램을 만들어보자.


리스트 연결

  • extend() : 리스트에 또 다른 리스트를 연결(확장)
  • 덧셈 연산자를 이용해서 리스트를 연결할 수도 있다.

[Ex] 나와 친구가 좋아하는 번호를 합치되 번호가 중복되지 않게 하는 프로그램을 만들자.


리스트 아이템 정렬

  • 오름차순 또는 내림차순으로 정렬
  • sort() : 오름차순 정렬
    ※ 내림차순의 경우, Option을 주어 변경 sort(reverse = True)

[Ex] 아래 점수표에서 최저 및 최고 점수를 삭제한 후 총점과 평균을 출력해보자


리스트 순서 뒤집기

  • reverse() : 아이템 순서를 뒤집을 수 있다.

[Ex] 다음은 전쟁에서 사용되는 암호이다. 암호를 해독하는 프로그램을 만들어보자.


리스트 슬라이싱

  • $[n:m]$을 이용하면 리스트에서 원하는 아이템만 뽑아낼 수 있다.
    • 앞 숫자($n$)를 생략하면, 처음부터 $m$까지 뽑아낼 수 있음
    • 뒷 숫자($m$)를 생략하면 $n$부터 끝까지 뽑아낼 수 있음
    • 음수를 사용하면, 뒤에서부터 계산


슬라이싱 단계 설정

  • 슬라이싱 할 때 단계를 설정할 수 있다.

numbers: [3, 5, 7, 9]

  • 슬라이싱을 이용해서 아이템을 변경할 수 있다.


slice()

  • slice() : 아이템을 슬라이싱하는 전용 함수
  • slice($A$, $B$) : $A$ 인덱스부터 $B$개 슬라이싱 한다.


리스트 나머지 기능

  • 리스트도 곱셈 연산이 가능
    • 특정 아이템의 인덱스를 찾을 수 있음
    • 특정 아이템의 갯수를 알아낼 수 있음
    • 필요없는 아이템은 삭제 가능
  • 리스트를 곱셈 연산 ▶ 아이템 반복
  • 함수를 통해 item의 인덱스를 알아낼 수 있다.
  • index(item) : item의 인덱스를 뽑아냄
  • index(item, $a$, $b$) : $a$부터 $b$ 앞까지 item의 인덱스 범위 설정 후, index 추출

[Ex] 1부터 10까지의 정수가 중복되지 않고 섞여있을 때 행운의 숫자 7의 위치를 찾자

  • count() : 특정 아이템의 갯수를 알아낼 수 있음
  • del() : 특정 아이템을 삭제
  • slicing을 이용하여 한 번에 여러 개 삭제 가능

[Ex] 하루 동안 헌혈을 진행한 후 혈액형 별 개수를 파악하는 프로그램을 만들어 보자.
