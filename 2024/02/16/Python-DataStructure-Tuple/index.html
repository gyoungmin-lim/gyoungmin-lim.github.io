(Python 중급) 자료구조 - 튜플(Tuple)

Created2024-02-16|Updated2024-02-17|StudyPython
|Post Views:

튜플(Tuple)

리스트(List)와 가장 큰 차이점

  • ※ 리스트(List)와 비슷하지만 아이템 변경 불가
  • 추가하려고 해 보면, 에러(Error)가 발생
  • 소괄호()를 이용해 선언하고, 데이터 구분은 콤마,를 이용
  • 숫자, 문자(열), 논리형 등 모든 기본 데이터를 같이 저장할 수 있음
  • 또 다른 컨테이너 자료형 데이터를 저장할 수도 있음

[Ex] 가족 이름을 튜플에 저장하고 출력해보자.

[Ex] 오늘 일정을 튜플에 저장하고 출력해보자.


튜플 아이템 조회

  • 인덱스를 이용하여 아이템 조회 가능

[Ex] 5명의 학생 이름을 튜플에 저장하고 인덱스가 홀수인 학생과 짝수(0 포함)인 학생을 구분해서 인덱스와 학생 이름을 출력해보자.

  • for문 사용


in/not in Keyword

  • 아이템 존재 유/무 판단
  • 문자열에서도 사용 가능

[Ex] 학생 유무 확인

[Ex] 문장에서 비속어가 있는지 알아내는 프로그램을 만들어보자.


튜플 길이

  • 튜플의 아이템 개수를 확인
  • 튜플 길이 : 튜플에 저장된 아이템 개수

[Ex] 좋아하는 운동 종목을 튜플에 저장하고 반복문을 이용해 출력해보자


튜플 결합

  • 덧셈 연산자 ‘+’ 를 이용해 두 개 이상의 튜플을 결합할 수 있다.
  • 리스트에서 사용할 수 있는 extend()함수를 튜플에서는 사용할 수 없다.

[Ex] 친구가 좋아하는 번호를 합치되 번호가 중복되지 않게 하는 프로그램을 만들자.


튜플 슬라이싱

  • 리스트와 마찬가지로 $[n:m]$을 이용해 원하는 아이템을 뽑아낼 수 있다.
  • 튜플은 슬라이싱을 이용해서 아이템을 변경할 수 없다.
  • slice() 함수를 이용해서 아이템을 슬라이싱할 수 있다.


리스트와 튜플

  • 튜플은 리스트와 달리 아이템 추가, 변경, 삭제가 불가능
  • 튜플은 선언 시 괄호 생략이 가능
  • 리스트와 튜플은 자료형 변환이 가능

[Ex] 튜플을 이용한 점수표에서 최저, 최고 점수를 삭제한 후, 총점과 평균을 출력해보자.


튜플 아이템 정렬

  • 튜플은 수정이 불가하기 때문에, 리스트 변환 후 정렬
  • sort() 함수를 이용하면 아이템을 정렬할 수 있다.
  • sorted() 함수를 정렬하면 튜플도 정렬할 수 있다.
    • sorted() : 튜플에서도 사용 가능한 오름차순 정렬. 반환되는 데이터는 ‘리스트’ 형식이며 원본은 유지됨


튜플과 for문

  • for문을 이용하여 튜플의 아이템을 자동으로 참조할 수 있다.
  • 튜플 내부에 또 다른 튜플의 아이템을 조회할 수도 있다.

[Ex] 학급별 학생 수와 전체 학생 수, 평균 학생 수를 출력해보자.

[Ex] for문과 if문을 이용해서 과락 과목 출력하기

[Ex] 사용자가 국어, 영어, 수학, 과학, 국사 점수를 입력하면 과락 과목가 점수를 출력하는 프로그램을 만들어보자.

[Ex] 학급 학생 수가 가장 작은 학급과 가장 많은 학급을 출력홰보자.


튜플과 While문

  • while문을 이용하여 다양한 방법으로 아이템 조회 가능

[Ex] While문과 if문을 이용해서 과락 과목 출력하기

[Ex] while문을 이용해서 학급 학생 수가 가장 적은 학급과 많은 학급을 출력해보자.
