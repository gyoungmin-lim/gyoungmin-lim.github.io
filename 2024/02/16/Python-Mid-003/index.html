(Python 중급) 텍스트(Text) 파일 다루기

Created2024-02-16|Updated2024-02-17|StudyPython
|Post Views:

텍스트 파일을 파이썬으로 다루기


텍스트 파일 쓰기

write()

  • 파일 쓰기
  • ‘w’ mode
    • 파일이 없으면 새로 생성
    • 기존에 있던 문자열을 다 없애버리고, 새로운 문자열로 작성
    • .write() : 괄호 내 해당 내용 작성

[Ex] 시스템 시간과 일정을 텍스트 파일에 작성해보자

  • 방법 1
  • 방법 2 : 모듈기능 이용
    • .strftime() : 형식에 해당하는 시간을 작성
기능 내용 기능 내용 기능 내용
%Y 연도 %H 시 %S a.m./p.m.
%m 월 %M 분
%d 날짜 %S 초


텍스트 파일 읽기

  • 텍스트 파일의 텍스트 읽기
  • read() 함수를 이용한 파일 문자열 읽기


read()

  • 파일 읽기
  • ‘r’ mode
    • 파일 읽기 기능을 가진 옵션(Option)
    • .read() : 해당 파일 내용 읽기(불러오기)

[Ex] 다음 텍스트 파일에서 ‘Python’을 ‘파이썬’으로 변경해서 파일에 다시 저장해보자.

.replace(A, B, C) : A 내용을 B 내용으로 C번 바꾸겠다.


텍스트 파일 열기

  • 파일을 다양한 방식으로 open할 수 있다.


open()

  • 목적에 따라 File mode를 선정한다.
File Mode 의 미
'w’ 쓰기 전용(파일이 있으면 덮어씌움)
'a’ 쓰기 전용(파일이 있으면 덧붙임)
‘x’ 쓰기 전용(파일이 있으면 에러 발생)
'r’ 읽기 전용(파일이 없으면 에러 발생)
  • ‘w’ Mode
  • ‘x’ Mode
  • ‘a’ Mode
  • ‘r’ Mode

[Ex] 사용자가 입력한 숫자에 대한 소수를 구하고 이를 파일에 작성해보자.

  • 파일 열기


close()

  • 파일 닫기


With ~ as문

  • 파일 닫기(Close)를 생략
  • Close 구문 없이 읽기(Read), 쓰기(Write) 가능

[Ex] 로또 번호 생성기 프로그램을 만들고 파일에 번호를 출력해보자


writelines()

  • 반복 가능한 자료형의 데이터를 파일에 작성
  • List 또는 Tuple 데이터를 파일에 쓰기 위한 함수
  • 딕셔너리에 저장된 과목별 점수를 파일에 저장하는 코드를 작성하자.


readlines()

  • 여러 줄 읽기
  • 파일의 모든 데이터를 읽어서 리스트 형태로 반환


readline()

  • 한 행을 읽어서 문자열로 반환

[Ex] 파일에 저장된 과목별 점수를 파이썬에서 읽어, 딕셔너리에 저장하는 코드를 만들어보자
