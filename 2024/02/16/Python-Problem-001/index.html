(Python 기초) 조건문, 반복문 문제

Created2024-02-16|Updated2024-02-16|StudyPython
|Post Views:

조건문 예제

[Ex] 상하수도 계산기

[Ex] 차량 운행제한 프로그램

  • 미세먼지 비상저감조치로 차량 운행제한 프로그램을 다음 내용에 맞게 만들어 보자.

[Ex] 난수 맞추기

  • PC에서 난수가 발생하면 사용자가 맞추는 게임을 만들어보자.


반복문 예제

[Ex] 홀/짝 구분하기

  • 1부터 100까지 정수 중 십의자리와 일의자리에 대해 각각 홀/짝수를 구분하는 프로그램을 만들어보자.

[Ex] 합, 홀수 합, 짝수 합, 팩토리얼 구하는 프로그램 만들기

  • 1부터 사용자가 입력한 정수까지의 합, 홀수의 합, 짝수의 합, 그리고 팩토리얼을 구하는 프로그램을 만들어보자.

[Ex] 반복문을 이용하여 프로그램의 실행 반복하기

[Ex] 버스정류장 버스 시간

  • 집 앞 버스 정류장에서 학교까지 가는 버스 A, B, C의 운행정보가 다음과 같을 때, 2대 이상의 버스가 정차하는 시간대를 출력해보자.
