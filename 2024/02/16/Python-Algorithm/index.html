(Python 중급) 알고리즘(Algorithm)

Created2024-02-16|Updated2024-02-17|StudyPython
|Post Views:

알고리즘(Algorithm)이란?

  • 어떠한 문제를 해결하기 위한 일련의 절차나 방법


검색 알고리즘

선형 검색

  • 선형으로 나열되어 있는 데이터를 순차적으로 스캔하면서 원하는 값을 찾음
  • 보초법 : 마지막 인덱스에 찾으려는 값을 추가해서 찾는 과정을 간략화

[Ex] 리스트에서 가장 앞에 있는 숫자 ‘7’을 검색하고 위치(인덱스)를 출력하자.

[Ex] 리스트에서 숫자 ‘7’을 모두 검색하고 각각의 위치(인덱스)와 검색 개수를 출력하자.

  • 알고리즘화


이진 검색

  • 가운데 데이터(중앙값) 이용
    ※ 정렬이 되어있는 상태라 가정함

  • 정렬되어 있는 자료구조에서 중앙값과의 크고 작음을 이용해서 데이터 검색

  • 검색 범위를 좁혀나가면서 원하는 값을 찾음


[Ex] 리스트를 오름차순으로 정렬 후 ‘7’을 검색하고 위치(인덱스)를 출력하자


순위(Rank)

  • 순위 : 수의 크고 작음을 이용해서 수의 순서를 정하는 것


[Ex] 학급 학생(20명)들의 중간고사와 기말고사 성적을 이용해서 각각의 순위를 구하고, 중간고사 대비 기말고사 순위 변화(편차)를 출력하는 프로그램을 만들어보자.(시험 성적은 난수 이용)

  • 클래스 생성
  • 실행 코드


정렬 알고리즘

버블 정렬

  • 처음부터 끝까지 차례로 비교
  • 처음부터 끝까지 인접하는 인덱스의 값을 순차적으로 비교하면서 큰 숫자를 가장 끝으로 옮기는 알고리즘


[Ex] 새 학년이 되어 학급에 20명의 새로운 학생들이 모였다. 학생들을 키 순서로 줄 세워 보자. 학생들의 키는 random 모듈을 이용해서 170~185 사이로 생성한다.

  • 모듈 생성
  • 실행 코드


삽입 정렬

  • 정렬되어 있는 부분에 들어가는 지 판단
  • 정렬되어 있는 자료 배열과 비교해서, 정렬 위치를 찾는다.

[Ex] 1부터 1000까지의 난수 100개를 생성하고, 다음의 요구 사항을 만족하는 모듈을 만들어보자.

  • 모듈 생성
  • 실행 코드


선택 정렬

  • 가장 작은 데이터를 찾아 자리 바꾸기
  • 주어진 리스트 중에 최솟값을 찾아 그 값을 맨 앞에 위치한 값과 교체하는 방식
  • 자료를 정렬하는 알고리즘


[Ex] 선택정렬 알고리즘을 이용해서 학생 20명의 시험 점수를 오름차순과 내림차순으로 정렬하는 모듈을 만들어보자. 시험 점수는 50점부터 100까지로 한다

  • 모듈 생성
  • 실행 코드


병합 정렬

  • 분할하고, 다시 합치기
  • 재귀 알고리즘 이용
  • 자료구조를 분할하고 각각의 분할된 자료구조를 정렬 후, 다시 병합하여 정렬

[Ex] 1부터 100까지의 난수를 생성하고, 다음의 요구 사항을 만족하는 모듈을 만들어보자.

요구사항 1. 병합정렬 알고리즘을 이용한 난수 정렬 모듈 구현
요구사항 2. 위의 모듈에 오름차순과 내림차순을 선택할 수 있는 옵션 추가

  • 모듈 생성
  • 실행 코드


퀵 정렬

  • 기준보다 작은 값과 큰 값을 분리
  • 기준 값보다 작은 값과 큰 값으로 분리한 후 다시 합침

[Ex] 1부터 100까지의 난수 10개를 생성하고, 다음의 요구사항을 만족하는 모듈을 만들어보자.

요구사항 1. 퀵정렬 알고리즘을 이용한 난수 정렬 모듈 구현
요구사항 2. 위의 모듈에 오름차순과 내림차순을 선택할 수 있는 옵션 추가

  • 모듈 생성
  • 실행 코드


수학 알고리즘

최댓값(Max)

  • 자료구조에서 가장 큰 값을 찾는다
  • ord() : 문자를 ASCII Code 값으로 반환

[Ex] 리스트에서 아스키코드가 가장 큰 값을 찾는 알고리즘을 만들어보자.


최솟값(Min)

  • 자료구조에서 가장 작은 값을 찾는다.

[Ex] 리스트에서 아스키코드가 가장 작은 값을 찾는 알고리즘을 만들어보자.


최빈값(Mid)

  • 최빈값 : 데이터에서 빈도수가 가장 많은 데이터

[Ex] 최빈값 알고리즘을 이용해서 학생 100명의 점수 분포를 다음과 같이 나타내 보자.

  • 모듈 생성
  • 실행 코드


근삿값(Approximation)

  • 근삿값 : 특정 값(참 값)에 가장 가까운 값

[Ex] 근삿값 알고리즘을 이용해서 시험 점수를 입력하면 학점이 출력되는 프로그램을 만들어보자.

  • 모듈 생성
  • 실행코드


평균(Mean)

  • 평균 : 여러 수나 양의 중간값을 갖는 수

[Ex] 50 이상 90이하 수들의 평균

[Ex] 정수들의 평균

[Ex] 평균을 구하고 순위를 정하는 알고리즘을 만들어보자.

  • 모듈 생성
  • 실행 코드


재귀(Recursive)

  • 재귀 : 나 자신을 다시 호출하는 것

[Ex] 재귀 알고리즘을 이용해 최대 공약수 계산

유클리드 호제법

두 자연수 $n1$, $n2$에 대하여 $(n1 > n2)$ $n1$를 $n2$로 나눈 나머지를 $r$이라고 할 때, $n1$과 $n2$의 최대공약수는 $n2$와 $r$의 최대공약수와 같다.


하노이의 탑

하노이의 탑이란?

  • 퍼즐 게임의 일종

  • 세 개의 기둥을 이용해서 원판을 다른 기둥으로 옮기면 되고, 제약 조건은 다음과 같음

    1. 한 번에 한 개의 원판만 옮길 수 있다.
    2. 큰 원판이 작은 원판 위에 있어서는 안된다.

  • 원판 갯수 2개

[Ex] 파이썬을 이용해서 하노이의 탑 게임 진행 과정을 출력하자
