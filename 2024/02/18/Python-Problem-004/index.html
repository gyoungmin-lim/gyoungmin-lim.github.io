(Python) 알고리즘(Algorithm) 문제

Created2024-02-18|Updated2024-02-18|StudyPython
|Post Views:

검색 알고리즘

선형검색

[Ex] 숫자로 이루어진 리스트에서 사용자가 입력한 숫자를 검색하는 모듈을 다음 요건에 따라 만들어 보자.

▶ 검색 모듈은 선형 검색 알고리즘을 이용
▶ 리스트는 1부터 20까지의 정수 중에서 난수 10개 이용
▶ 검색 과정을 로그로 출력
▶ 검색에 성공하면 해당 정수의 인덱스 출력, 결과가 없으면 -1 출력

  • 모듈 생성
  • 실행 코드


이진검색

  • [Recall] 데이터가 정렬이 되어있는 상태여야 함

[Ex] 숫자로 이루어진 리스트에서 사용자가 입력한 숫자를 검색하는 모듈을 다음 요건에 따라 만들어보자.

▶ 검색 모듈은 이진 검색 알고리즘을 이용
▶ 리스트는 [1, 2, 4, 6, 7, 8, 10, 11, 13, 15, 16, 17, 20, 21, 23, 24, 27, 28] 이용
▶ 검색 과정을 로그로 출력
▶ 검색에 성공하면 해당 정수의 인덱스 출력, 결과가 없으면 -1 출력

  • 모듈 생성
  • 실행 코드


순위 알고리즘

[Ex] 숫자로 이루어진 리스트에서 아이템의 순위를 출력하고, 순위에 따라 아이템을 정렬하는 모듈을 만들어보자. 리스트는 50부터 100까지의 난수 20개를 이용하자.

  • 모듈 생성
  • 실행코드

[Ex] 알파벳 문자들과 정수들에 대한 순위를 정하는 프로그램을 순위 알고리즘을 이용해서 만들어보자. 단, 알파벳은 아스키코드 값을 이용한다.


정렬 알고리즘

버블정렬

[Ex] 숫자로 이루어진 리스트를 버블정렬 알고리즘을 이용해서 오름차순과 내림차순으로 정렬하는 모듈을 만들어보자. (단, 정렬하는 과정도 출력하도록 한다)

  • 모듈생성
  • 실행코드


삽입정렬

[Ex] 숫자로 이루어진 리스트를 삽입정렬 알고리즘을 이용해서 오름차순과 내림차순으로 정렬하는 모듈을 만들어보자.

  • 모듈 생성
  • 실행코드


선택정렬

[Ex] 숫자로 이루어진 리스트를 선택정렬 알고리즘을 이용해서 오름차순과 내림차순으로 정렬하는 모듈을 만들어보자. (단, 정렬하는 과정도 출력하도록 한다.)

  • 모듈 생성
  • 실행코드


병합정렬

[Ex] 숫자로 이루어진 리스트를 병합정렬 알고리즘을 이용해서 오름차순과 내림차순으로 정렬하는 모듈을 만들어보자.(단, 정렬하는 과정도 출력하도록 한다.)

  • 모듈 생성
  • 실행코드


수학 알고리즘

최댓값

[Ex] 최댓값 알고리즘을 이용해서 숫자로 이루어진 리스트에서 최댓값과 최댓값의 갯수를 찾는 모듈을 만들어보자. (리스트는 1부터 50까지의 난수 30개를 이용하되, 중복이 허용되도록 한다.)

  • 클래스 생성
  • 실행코드


[Ex] 학급 전체 학생의 시험 점수에 대한 평균과 최댓값을 구하고 평균과 최댓값의 편차를 출력하는 프로그램을 최댓값 알고리즘을 이용해서 만들어보자.

  • 모듈 생성
  • 실행코드


최솟값

[Ex] 최솟값 알고리즘을 이용해서 숫자로 이루어진 리스트에서 최솟값과 최솟값의 갯수를 찾는 모듈을 만들어보자. (리스트는 1부터 50까지의 난수 30개를 이용하되, 중복이 허용되도록 한다.)

  • 클래스 생성
  • 실행코드

[Ex] 학급 전체 학생의 시험 점수에 대한 평균과 최솟값을 구하고 평균과 최솟값의 편차를 출력하는 프로그램을 최솟값 알고리즘을 이용해서 만들어보자.

방법 1

  • 모듈 생성
  • 실행코드

방법 2

  • 클래스(Class) 생성
  • 실행코드


최빈값

[Ex] 다음은 어떤 회사의 전직원 나이를 나타내는 리스트이다. 최빈값 알고리즘을 이용해서 나이 분포를 간단한 그래프로 출력하는 모듈을 만들어보자.

  • max class 생성
  • mod class 생성
  • 실행코드

[Ex] 다음은 회차별 로또 번호이다. 최빈도 알고리즘을 이용해서 모든 회차의 각각의 번호에 대한 빈도수를 출력하는 프로그램을 만들어보자.

  • 클래스 생성
  • 실행코드


근삿값

[Ex] 다음 표는 수심에 따른 수온을 나타내고 있다. 근삿값 알고리즘을 이용해서 수심을 입력하면 수온을 출력하는 모듈을 만들어보자.

  • 클래스 생성
  • 실행코드

[Ex] 사용자의 몸무게(kg)와 키(m)를 입력하면 체질량지수(BMI)를 계산하고, 근삿값 알고리즘과 BMI 표를 이용해서 신체 상태를 출력하는 프로그램을 만들어보자.

  • 클래스 생성
  • 실행코드


재귀 알고리즘

[Ex] 다음은 ‘A상사’의 2021년 월별 매출을 나타내는 표이다. 재귀 알고리즘을 이용해서 1월부터 12월까지 전월대비 매출 증감액을 나타내는 프로그램을 만들어보자.


[Ex] 사용자가 정수 두개를 입력하면 작은 정수와 큰 정수 사이의 모든 정수의 합을 구하는 프로그램을 재귀 알고리즘을 이용해서 만들어보자.

  • 클래스 생성
  • 실행코드


평균 알고리즘

[Ex] 다음은 어떤 체조선수의 경기 점수이다. 최댓값과 최솟값을 제외한 나머지 점수에 대한 평균을 구하고 순위를 정하는 알고리즘을 만들어보자.

  • 최대점수 관련 클래스 생성
  • 최소점수 관련 클래스 생성
  • Top5 점수 관련 클래스 생성 (Near Algorithm)
  • 실행코드


[Ex] 다음은 홍길동학생을 포함한 학급 전체 학생의 시험 점수 평균을 나타낸 표이다. 표를 보고, 홍길동 학생을 제외한 나머지 학생의 평균과 홍길동 학생의 점수의 차이를 출력하는 프로그램을 만들어보자. (출력은 과목별 점수와 평균 점수를 모두 출력한다)
