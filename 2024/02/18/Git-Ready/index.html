참고 - Line ending

  • Windows : CR (\r) + LF (\n)
  • Unix or Mac : LF (\n)
  • Windows 사용자와 Mac 사용자가 같은 Git Repository를 작업할 때 코드에서 변경된 내용이 없어도 CRLF 차이로 인해 commit이 발생할 수 있음
  • CRLF - Windows
    • Windows - 가져올 때는 LF를 CRLF로 변경하고, 보낼 때는 CRLF를 LF로 변경
  • CRLF - Mac
    • Mac - LF만 사용
