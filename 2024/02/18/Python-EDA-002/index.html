  • 범죄별 연관성 확인


Data

  • 범죄데이터
    • 공공데이터 포털
    • 서울시 5대 범죄 발생현황 통계


Preprocessing

  • ModuleLoad

Crime in Seoul

  • DataLoad
    • 숫자값들이 콤마(,)를 사용하고 있어 문자로 인식될 수 있다.
    • 천 단위 구분(thousands = ‘,’)이라고 알려주면 콤마를 제거하고 숫자형으로 읽는다
  • 범죄 종류 확인

Output : array(['살인', '강도', '강간', '절도', '폭력', nan], dtype=object)

  • NaN 값 제거
  • 제거 전과 제거 후 비교
제거 전 제거 후
  • 피벗테이블을 통해 구별 범죄 발생 수를 표현
  • column 구성 확인
    • 피벗테이블의 경우, column이 multi로 잡힘 (단점)
  • 필요없는 column level 삭제
    • columns Level은 위에서부터 0, 1, 2 ~ 순서
  • 테이블의 index 확인
    • 구분 인덱스의 경우 경찰서 이름으로 되어있다.
    • 경찰서 이름으로 구 이름을 알아야 함


GoogleMaps를 이용한 데이터 정리

  • ModuleLoad & GoogleKey
  • gmaps.geocode() 통해 들어오는 정보 확인
  • gmaps.geocode() 를 통해 좌표(lat, lng)와 주소(address) 호출 여부 확인

Output : 37.5260441, 126.9008091, 대한민국 서울특별시 영등포구 국회대로 608

  • 지역 구에 해당하는 부분 추출

Output : '영등포구’

  • 추출 데이터를 저장하기 위해, 기존 데이터프레임(DataFrame)에 새 컬럼(Columns) 생성
    • 구별, lat, lng column 추가
  • 경찰서 이름에서 소속된 구 이름 얻기
    • iterrow() 함수 사용
    • 구 이름을 얻기 위해, 경찰서 주소정보를 가져옴
  • 구 이름과 위도(lat), 경도(lon) 정보를 저장
    • for문을 이용해서 위 표의 NaN을 모두 채워줌
  • Column명 변경
    • 0 level, 1 level column을 통해 데이터프레임(DataFrame) Column명 재설정

Save Data


구별 데이터로 정리

  • 구별 데이터로 정리하기 위해 pivot_table
  • 각 범죄별 발생 대비 검거비율을 보기 위해, 새로운 column에 계산하여 표기
  • 발생 건수와 검거율에 대해 볼 것이기 때문에, 검거 건수는 불필요. 따라서 column 제거
  • 범죄검거율의 경우, 100%를 초과하는 것이 발생
    • 이전에 발생하였던 사건이 그 다음에 해결되어 카운트(count) 되었을 가능성이 있다.
      • ex) 7월에 사건 발생 > 8월에 사건 해결
    • 100%를 초과할 경우, 100%로 조정
  • column명 간소화


최종 정리

  • 정규화 데이터 생성
    • 범죄 건수의 단위 차이가 많이나기 때문에, 이를 정규화(normalization)하여 표현
  • 정규화(normalization)된 데이터와 검거율을 하나의 데이터프레임(DataFrame)으로 저장

  • CCTV 및 인구 데이터를 불러와 붙여넣기

    • DataLoad

    • 데이터 붙여넣기

  • 정규화된 5개 범죄 발생 평균과 검거율 평균을 새 column에 추가

  • CCTV 설치 갯수 및 인구수와 범죄(폭력, 살인) 검거율 간 연관성 확인

서울시 범죄율에 대한 지도 시각화

LoadPath

ModuleLoad

DataLoad

  • 서울특별시 지도 좌표 설정 및 불러오기

  • ※ 지도에 검은색 배경으로 나타나는 이유
    ➣ folium 공식 문서에 의하면 choropleth 객체에 주게 되는 인자들 중 nan_fill_color는 디폴트로 검은색이 적용

  • 정규화된 살인 발생 건수를 지도에 시각화

  • 같은 방법으로, 정규화된 강간 발생 건수를 지도에 시각화
  • 5대 범죄 발생 건수를 시각화
  • 인구 대비 범죄 발생 건수 시각화

  • 모든 범죄에 대한 검거 건수를 정규화(normalization)하여 평균으로 계산
  • 경찰서 위치 Marker 표시
  • 검거 column에 값을 곱한 뒤, 원의 넓이 적용
    • 검거는 범죄 검거수를 기준으로 정규화(normalization) 되었으므로 검거 수가 높은 구를 표현하기 위함


서울시 범죄현황 발생 장소 분석

  • DataLoad
  • 범죄 발생 장소 확인

Output : array(['아파트, 연립 다세대', '단독주택', '노상', '상점', '숙박업소, 목욕탕', '유흥 접객업소', '사무실','역, 대합실', '교통수단', '유원지 ', '학교', '금융기관', '기타'], dtype=object)

  • 범죄 발생 장소를 기준으로 각 범죄 발생 건수 합계를 계산

  • 정규화된 값을 기준으로, 각 범죄의 평균 발생 건수를 구함
    • 총합 column에 저장
