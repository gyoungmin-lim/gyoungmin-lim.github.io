(Python) 판다스(Pandas) 기초, 리스트(List) 데이터형 처리 및 Numpy Package

Created2024-02-18|Updated2024-02-18|ToolPython
|Post Views:

Pandas 기초

  • Python에서 R 만큼의 강력한 데이터 핸들링 성능을 제공하는 모듈
  • 단일 프로세스에서는 최대 효율
  • 코딩 가능하고 응용 가능한 엑셀로 받아들여도 됨
  • 누군가 스테로이드를 맞은 엑셀로 표현함


Series

  • index와 value로 이루어짐
  • 한 가지 데이터타입만 가질 수 있음


DataFrame

  • pd.Series()
    • index, value
  • pd.DataFrame()
    • index, value, column


자주 사용하는 명령어

  • 데이터 정보 탐색할 때 많이 사용

  • df.head()

    • 상위 5개 데이터 추출
  • df.tail()
    • 하위 5개 데이터 추출
  • df.index
    • 인덱스별 데이터 정보 확인
  • df.columns
    • DataFrame의 column명 확인
  • df.values
    • DataFrame의 value 확인
  • df.info()
    • DataFrame의 기본 정보 확인
  • df.describe()
    • DataFrame의 기술통계 정보 확인
    • 데이터의 갯수, 평균, 표준편차, 최댓값, 최솟값, 4분위수 확인


데이터 정렬

  • sort_values(by =’컬럼명’, ascending = True/False, inplace = True)
    • 특정 column(열) 기준으로 데이터를 정렬


데이터 선택

  • offset index
    • $[n:m]$ : $n$부터 $m-1$까지
    • 인덱스나 컬럼의 이름으로 slice하는 경우, 끝을 포함
  • df.loc
    • loc : location
    • idex 이름으로 특정 행,열을 선택
  • df.iloc
    • iloc : inter location
    • 컴퓨터가 인식하는 인덱스 값으로 선택


조건(condition)

  • A 컬럼에서 0보다 큰 숫자(양수)만 선택
  • NaN : not a Number
  • df.isin()
    • 특정 요소가 있는지 확인
    • True/False로 반환


특정 Columns 제거

  • `del df`
  • df.drop('컬럼명', axis = 0 or 1)
    • axis = 0 : 가로
    • axis = 1 : 세로


apply()

  • DataFrame에 일괄적으로 함수를 적용해주는 명령어
    • df[”컬럼명”].apply(”기능”)
    • 예제
      • df[”A”].apply(”sum”)
      • df[”A”].apply(”mean”)
      • df[”A”].apply(np.std)
      • df[”A”].apply(np.sum)


데이터 병합

  • pd.concat()

  • pd.merge()

    • 두 데이터 프레임에서 컬럼이나 인덱스를 기준으로 잡고 병합하는 방법

    • 키(Key)값 : 기준이 되는 컬럼이나 인덱스

      ※ 두 데이터프레임에 모두 포함되어 있어야 함

    • pd.merge(데이터프레임1, 데이터프레임2, how = ‘기능’, on = ‘key’)

      • on = ‘key’ : key column을 기준으로 병합
      • how = ‘기능’ : 데이터 병함을 데이터프레임1 기준으로 어떻게 병합 할 것인지
        • left : 데이터프레임1 Key값을 기준으로 데이터 병합
        • right : 데이터프레임2 Key값을 기준으로 데이터 병합
        • outer : 둘 다 손상되지 않도록 병합
        • inner : 데이터프레임의 공통 데이터만 병합
  • pd.join()


인덱스 변경

  • set_index()
  • 선택한 column을 데이터 프레임의 인덱스로 지정


상관계수

  • corr() : correlation
  • 보통 상관계수가 0.2 이상인 데이터를 비교


Numpy

1차 직선 만들기

  • np.polyfit
    • 직선을 구성하기 위한 계수 계산
  • np.poly1d
    • polyfit으로 찾은 계수로 python에서 사용할 함수로 만들어 줌


Pivot_Table

iterrows()

  • Pandas 데이터 프레임은 대부분 2차원
  • for문 사용 시 ‘$n$번째’라는 지정을 반복해서 가독률 떨어짐
  • Pandas 데이터 프레임으로 반복문을 만들 때 iterrows()옵션을 사용하면 편함
  • 받을 때, 인덱스와 내용으로 나누어 받는 것 주의


List Data형

기본 형태

  • list형은 대괄호로 생성

Output : ('red', 'blue', 'green')

Output : ('red', 'blue', 'green')

Output : ['red', 'blue', 'green']

Output : ['red', 'black', 'green']

  • 깊은복사 : .copy() 활용

Output : ['red', 'black', 'green']

Output : ['red', 'yellow', 'green']

Output : ['red', 'black', 'green']


반복문

  • List 형을 반복문(for)에서 사용하는 방법이 다른 언어에 비해 편리함


조건문

  • in 명령으로 조건문(if)에서 사용하는 방법이 다른 언어에 비해 편리함


다양한 기능들

Output : ['라라랜드', '먼 훗날 우리', '어벤저스', '다크나이트']

  • append: list 제일 뒤에 추가

Output : ['라라랜드', '먼 훗날 우리', '어벤저스', '다크나이트', '타이타닉']

  • pop: 리스트 제일 뒤부터 자료를 하나씩 삭제

Output : ['라라랜드', '먼 훗날 우리', '어벤저스', '다크나이트']

  • extend: 제일 뒤에 자료 추가

Output : ['라라랜드', '먼 훗날 우리', '어벤저스', '다크나이트', '위대한쇼맨', '인셉션', '터미네이터']

  • remove: 자료를 삭제

Output : ['라라랜드', '먼 훗날 우리', '다크나이트', '위대한쇼맨', '인셉션', '터미네이터']

  • 슬라이싱: $[n:m]$ $n$번째부터 $m-1$번째 까지

Output : ['위대한쇼맨', '인셉션']

  • insert: 원하는 위치에 자료를 삽입

Output : ['위대한쇼맨', 9.6, '인셉션']

Output : ['위대한쇼맨', 9.6, '인셉션', 9.5]

  • list안에 list 형태 가능

Output : ['위대한쇼맨', 9.6, '인셉션', 9.5, ['레오나르도 디카프리오', '조용하']]

  • isinstance: 자료형 True/False

Output : True
