(Python) 자료구조(Data Structure) 문제

Created2024-02-18|Updated2024-02-18|StudyPython
|Post Views:

리스트(List)

[Ex] 1부터 사용자가 입력한 숫자까지의 약수와 소수를 리스트에 각각 저장하고, 이것을 출력하는 프로그램을 만들어보자.

[Ex] 1부터 100 사이에 난수 10개를 생성한 후 짝수와 홀수를 구분해서 리스트에 저장하고 각각의 갯수를 출력하는 프로그램을 만들어보자.

[Ex] 다음은 공원 입장료를 나타낸 표이다. 1일 총 입장객이 100명이라고 할 때, 1일 전체 입장 요금을 구하는 프로그램을 만들어보자. (단, 입장 고객의 나이는 난수를 이용한다.)

[Ex] 친구 이름 다섯명을 리스트에 저장하고 오름차순과 내림차순으로 정렬해보자.

[Ex] 다음 리스트에서 중복 아이템(숫자)을 제거하는 프로그램을 만들어보자

[Ex] 4개의 숫자 중 서로 다른 숫자 2개를 선택해서 만들 수 있는 모든 경우의 수를 출력하는 프로그램을 만들어보자.

[Ex] 4개의 숫자 중 서로 다른 숫자 3개를 선택해서 만들 수 있는 모든 경우의 수를 출력하는 프로그램을 만들어보자.


튜플(Tuple)

[Ex] 자주 접속하는 웹사이트 비번을 튜플에 저장해보자.

[Ex] 대학생 길동이의 1, 2, 3학년의 성적은 다음과 같다. 졸업할 때 4.0이상의 학점을 받기 위해 긷롱이가 받아야하는 4학년 1, 2학기의 최소 학점을 구해보자.

[Ex] 2개의 튜플에 대해서 합집합과 교집합을 출력해보자.

[Ex] 튜플을 요구사항에 맞춰 아이템을 슬라이스하자.

요구사항

  • Index 0부터 3까지
  • Index 2부터 4까지
  • Index 3부터 끝까지
  • Index 2부터 뒤에서 -2까지
  • Index 0부터 끝까지 3단계

[Ex] 시험점수를 입력한 후 튜플에 저장하고 과목별 학점을 출력해보자.

[Ex] 튜플의 과일 개수에 대해서 오름차순 및 내림차순으로 정렬해보자.

[Ex] 학급별 학생 수를 나타낸 튜플을 이용해서 요구사항에 맞는 데이터를 출력하는 프로그램을 만들어보자.

요구사항

  • 전체 학생 수
  • 평균 학생 수
  • 학생 수가 가장 적은 학급
  • 학생 수가 가장 많은 학급
  • 학급별 학생 편차


딕셔너리(Dictionary)

[Ex] 과목별 점수를 딕셔너리에 저장하고 출력하는 프로그램을 만들어보자.

[Ex] 삼각형부터 십각형까지의 내각의 합과 내각을 딕셔너리에 저장하는 프로그램을 만들어보자.

[Ex] 1부터 10까지의 각각의 정수에 대한 약수를 저장하는 딕셔너리를 만들고 출력하는 프로그램을 만들어보자.

[Ex] 다음 문구를 공백으로 구분하여 리스트에 저장한 후, 인덱스와 단어를 이용해서 딕셔너리에 저장해보자.

aboutPython = ‘파이썬은 1991년 프로그래머인 귀도 반 로섬이 발표한 고급 프로그래밍 언어이다.’

[Ex] 다음 문장에서 비속어를 찾고 비속어를 표준어로 변경하는 프로그램을 만들어보자.

‘강도는 서로 쪼개다, 짭새를 보고 빠르게 따돌리며 먹튀했다.’

[Ex] 딕셔너리를 이용해서 5명의 회원을 가입 받고 전체 회원 정보를 출력하는 프로그램을 만들어보자.

[Ex] 위의 프로그램을 이용해서 특정 회원 계정을 삭제하는 프로그램을 만들어보자
