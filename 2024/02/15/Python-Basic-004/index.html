(Python 기초) 기초수학

Created2024-02-15|Updated2024-02-17|StudyPython
|Post Views:

약수

  • 어떤 수를 나누어 떨어지게 하는 수


  • 1과 그 자신만을 약수로 가지는 수(단, 1은 제회)


소인수

  • 약수(인수) 중에서 소수인 숫자

  • 소인수 예제

    • 12의 소인수 : [2, 3]
    • 36의 소인수 : [2, 3]
    • 25의 소인수 : [5]
    • 41의 소인수 : [41]


소인수 분해

  • 1보다 큰 정수를 소인수의 곱으로 나타낸 것

[Ex] 소인수 찾기

[Ex] 72에 x를 곱하면 y의 제곱이 된다고 할 때, x에 해당하는 가장 작은 정수를 구하자


최대공약수

  • 공통된 약수 중에서 가장 큰 수
  • 소인수 분해를 이용하면 최대공약수 및 공약수를 구할 수 있다.
    ※ 소수로 나눗셈하면 좀 더 편리하게 최대공약수를 구할 수 있음


공약수

  • 두 개 이상의 수에서 공통된 약수
  • 공약수 중 가장 큰 수를 최대공약수라고 한다.

[Ex] 두 수를 입력하면 공약수와 최대공약수를 출력

[Ex] 유클리드 호제법을 이용해서 최대공약수 구하기


최소공배수

  • 공통된 배수 중 가장 작은 수


공배수

  • 두 개 이상의 수에서 공통된 배수
  • 소인수분해를 이용하면 최소공배수 및 공배수를 구할 수 있다.
    ※ 소수로 나눗셈하면 좀 더 편리하게 구할 수 있음

[Ex] 두 수를 입력하면 최소공배수를 출력하는 코드를 작성하자.

[Ex] 세 가지 수의 최소공배수

  • 섬마을에 과일, 생선, 야채를 판매하는 배가 다음 주기로 입항한다고 할 때, 모든 배가 함께 입항하는 날짜를 계산해보자
    • 과일 선박 : 3일주기, 생선선박 4일 주기, 야채 선박 : 5일 주기


진법

  • 진법이란, 특정 숫자 몇 개를 사용하여 수를 표시하는 방법
  • 대표적으로 2진법, 8진법, 10진법, 16진법이 존재
    • 2진수 binary : bin() ‘#b’
    • 8진수 octal : oct() ‘#o’
    • 10진수 integer : int()
    • 16진수 hexadecimal : hex() ‘#x’
  • Type : 문자열(Str)

[Ex] 10진수를 2진수, 8진수, 16진수로 변환

  • 방법 1
  • 방법 2
  • 방법 3

[Ex] X진수를 10진수로 변환


수열

  • 규칙성을 가지고 나열되어있는 수


등차수열

  • 연속하는 두 수의 차이가 모두 일정한 수열
  • $a_n$ : $n$번째 항, $d$ : 공차라고 하자.

[Ex] 등차수열의 $n$번째 항을 구하는 코드 작성

  • 방법 1
  • 방법 2(공식 사용)
  • 등차 중항 : 연속된 세 항에서 가운데 항
  • 등차수열의 합

[Ex] 등차수열의 합을 구하는 코드 작성

  • 방법 1
  • 방법 2(공식 사용)


등비수열

  • 각 항이 초항(First term)과 일정한 비를 가지는 수열
  • 초항을 $a$라 하고, 공비를 $r$이라 하자.

[Ex] 등비수열의 $n$번째 항을 구하는 코드 작성

  • 방법 1
  • 방법 2 (공식 사용)
  • 등비 중항 : 연속된 세 항에서 가운데 항
  • 등비수열의 합

[Ex] 등비수열의 합을 구하는 코드 작성

  • 방법 1
  • 방법 2(공식 사용)


계차수열

  • 어떤 수열 ${a_n}$의 인접하는 두 항의 차로 이루어진 또 다른 수열
  • 계차 수열 $b_n$의 일반항

[Ex] 계차수열 $n$번째 항 구하기

  • 방법 1
  • 방법 2(공식 사용)


피보나치 수열

  • 앞의 두 항을 더하여 만든 값
  • 토끼의 번식

[Ex] 피보나치 수열 코드


시그마

  • 수열의 합을 나타낸 기호 $\sum$
  • $n$항까지의 합을 구할 때 사용


팩토리얼

  • 1부터 양의 정수 $n$까지의 정수를 모두 곱한 것

[Ex] Python을 이용해서 팩토리얼을 결과값 출력

  • 방법 1. For문 사용
  • 방법 2. While문 사용
  • 방법 3. 함수 사용
  • 방법 4. Math module 사용


군 수열

  • 여러 개의 항을 묶었을 때 규칙성을 가지는 수열

[Ex] 다음 수열을 보고 $n$번째 항의 값을 출력하는 프로그램을 만들어 보자


순열(Permutation)

  • 일렬로 나열하는 경우의 수
  • $n$개에서 $r$개를 택하여 나열하는 경우의 수
  • 순열은 팩토리얼(계승)을 이용해서 나타낼 수 있다.

[Ex] 다음 순열들의 값을 구하는 프로그램을 만들어 보자.


원순열

  • 시작과 끝의 구분이 없는 순열

[Ex] 4명의 친구가 원탁 테이블에 앉을 수 있는 순서를 계산해보자.


조합(Combination)

  • $n$개에서 $r$개를 택하는 경우의 수
    ※ 순열과의 차이점 : 순열은 순서가 중요!

[Ex] Python을 사용해서 다음 조합의 값을 구하는 프로그램을 만들어 보자.

[Ex] 카드 5장 중 2장을 선택했을 때 삼각형과 사각형이 동시에 선택될 수 있는 확륭른?


확률(Probability)

  • 특정 사건이 일어날 수 있는 수

  • 확률 : 모든 사건에서 특정 사건이 일어날 수 있는 수를 나타낸 것

  • 모든 사건을 Sample space, 특정 사건은 Event라고도 함

동전 던지기

  • 모든 사건 : 앞, 뒤
  • 특정 사건 : 앞(1가지), 뒤(1가지)
  • 확률 : 앞($\frac{1}{2}$), 뒤 : 뒤()

주사위 굴리기

  • 모든 사건 : 1, 2, 3, 4, 5, 6
  • 특정 사건 : 1 (1가지), 2 (1가지), 3 (1가지), 4 (1가지), 5 (1가지), 6 (1가지)
  • 확률 : 1($\frac{1}{6}$), 2($\frac{1}{6}$), 3($\frac{1}{6}$), 4($\frac{1}{6}$), 5($\frac{1}{6}$), 6($\frac{1}{6}$)
  • 조합을 이용해 확률을 알아낼 수 있다.


[Ex] 박스에 ‘꽝’이 적힌 종이가 4장 있고, ‘선물’이 적힌 종이가 3장이 있을 때, 파이썬을 이용해 ‘꽝’ 2장과 ‘선물’ 1장을 뽑는 확률(%)을 출력
