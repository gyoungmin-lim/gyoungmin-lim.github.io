(MySQL) 데이터 조작어(Data Manipulation Language)

Created2024-02-15|Updated2024-02-15|StudyMySQL
|Post Views:

실습환경 만들기

  • person table 생성


INSERT

Data 추가

  • 입력한 Column 이름의 순서와 값의 순서가 일치하도록 주의


모든 Column 값을 추가하는 경우

  • Column 이름을 지정하지 않아도 가능
  • 입력하는 값의 순서가 테이블 Column 순서와 일치하도록 주의할 것


SELECT

데이터를 조회하는 명령어

  • 테이블 내의 특정 칼럼에 대한 데이터를 조회

[Ex] person 테이블 내의 이름, 나이, 성별 데이터를 조회

  • 테이블 내의 모든 컬럼에 대한 데이터를 조회


WHERE

SQL문에 조건을 추가하여 SELECT 뿐만 아니라 UPDATE, DELETE에도 사용

  • 테이블 내에서 조건을 만족하는 데이터 조회

[Ex] person 테이블에서 성별이 여자인 데이터 조회


UPDATE

데이터 수정

[Ex] ‘이효리’ 나이를 23세로 수정


DELETE

데이터 삭제

[Ex] 이상순 데이터 삭제




실습환경 만들기

  • 데이터 추가


ORDER BY

  • SELECT 문에서 데이터를 특정 컬럼을 기준으로 오름차순 혹은 내림차순 정렬
    • ASC(Ascending) : 오름차순 정렬
    • DESC(Descending) : 내림차순 정렬
    • 기본 Option : 오름차순 정렬

[Ex] celeb 테이블에서 이름과 나이를 나이순으로 조회

[Ex] celeb 테이블에서 이름과 나이를 나이 역순(내림차순)으로 조회

[Ex] celeb 테이블에서 이름과 나이를 나이와 이름순으로 정렬하여 조회

[Ex] celeb 테이블에서 이름과 나이를 나이의 역순으로 정렬한 뒤, 이름순으로 정렬하여 조회

[Ex] celeb 테이블에서 이름, 생년월일, 성별, 소속자 데이터를 소속사 순으로 정렬하여 조회

[Ex] celeb 테이블에서 전체 컬럼을 전체 소속사, 이름 순으로 정렬하여 조회

[Ex] celeb 테이블에서 이름, 나이, 직업, 소속사 데이터를 소속사 순, 나이 역순으로 정렬하여 조회
