(Python 중급) 함수(Function), 변수(Variable), 모듈(Module), 패키지(Package)

Created2024-02-15|Updated2024-02-17|StudyPython
|Post Views:

함수란?

  • 파이썬의 함수는 수학의 함수와 동일
  • 함수 사용 이유 : 특정 기능을 재사용하기 위해 기능을 함수로 정의하여 사용


내장함수, 사용자함수

  • 함수는 파이썬에서 기본으로 제공하는 내장함수와 사용자가 직접 선언하는 사용자 함수가 존재




함수 선언

함수는 def 키워드, 함수명, : 들여쓰기를 이용하여 선언함

  • 첫 글자는 소문자가 관례 / 숫자나 특수문자는 쓸 수 없음
  • 변수는 보통 명사, 함수는 동사로 작성
    • 함수의 의미를 알아볼 수 있게 작성
  • 실행 부분은 들여쓰기 반드시!
    • 들여쓰기가 없을 경우, 에러가 발생할 수 있음

[Ex] 두 수의 덧셈

[Ex] 곱셈, 나눗셈 출력하기

  • 정수 두개를 입력하면 곱셈과 나눗셈 연산 결과를 출력하는 함수를 만들고 호출해보자.


함수 내 다른 함수 호출

함수 내에서 또 다른 함수를 호출할 수 있다.

  • pass를 이용해 실행물을 생략할 수 있음
    • 실행문이 정해지지 않았을 때 유용

[Ex] 구구단 출력 함수가 연쇄적으로 호출되도록 함수를 선언해보자.


인수와 매개변수

함수 호출 시 함수에 데이터를 전달할 수 있다

  • 함수 작성 시, 데이터를 넣었을 때, 함수 내 ‘매개변수’에 할당되는 것을 이용
    • 이 때, 함수에 주는 데이터를 ‘인수’ 또는 ‘파라미터’라고 함
    • 즉, 호출부와 선언부의 관계를 맺어주는 것
  • 인수와 매개변수 개수는 일치해야 한다.
  • 매개변수 개수가 정해지지 않은 경우 *를 이용


함수 실행결과 반환

  • return 키워드 이용, 함수 실행결과를 호출부로 반환함
  • 함수가 return을 만나면 실행을 종료
    • return 이후 코드내용은 실행하지 않는다.

[Ex] 홀, 짝 구분하기

[Ex] cm을 mm으로 환산하기

  • 사용자가 길이(cm)를 입력하면 mm로 환산한 값을 반환하는 함수를 만들어보자

[Ex] 1부터 100까지의 정수 중에서 홀수인 난수를 반환하는 함수를 선언하자.


지역변수와 전역변수

전역변수

  • 함수 밖에 선언된 변수
  • 어디에서나 사용 가능하나, 함수 안에서 수정할 수 없음


지역변수

  • 함수 안에 선언된 변수
  • 함수 안에서만 사용 가능


Global Keyword

  • Global을 사용하면 함수 안에서도 전역변수의 값을 수정할 수 있음

[Ex] 삼각형, 사각형 넓이 구하기

  • 사용자가 가로, 세로 길이를 입력하면 삼각형과 사각형의 넓이를 구하는 함수를 만들어보자.


중첩함수

  • 함수 안에 또 다른 형수가 있는 형태
    • 함수 안에서 호출해서 사용 가능
    • 내부 함수를 함수 밖에서 호출할 수 없음 (함수 내에서만 사용 가능)

[Ex] 계산기


Lambda 함수

함수 선언을 보다 간단하게 하자!

  • lambda 변수1, 변수2, $\cdots$ : 함수식 형태

  • 기존 함수 정의

  • Lambda 함수 정의

[Ex] lambda 함수 이용 삼각형, 사각형, 원의 넓이 구하기




모듈(Module)

모듈이란?

  • 이미 만들어진 훌륭한 기능

  • 사용자는 쉽게 사용할 수 있다.

    • 계산 모듈, 난수모듈, 날짜/시간 모듈 등

  • Python Module은 내부, 외부, 사용자 모듈로 구분

    • 내부 모듈 : 파이썬 설치 시 기본적으로 사용할 수 있는 모듈
    • 외부 모듈 : 별도 설치 후 사용할 수 있는 모듈(Pandas, Numpy)
    • 사용자 모듈 : 사용자가 직접 만든 모듈


모듈(Module) 제작

모듈은 파이썬 파일이다!

  • 모듈(Module)은 특정 기능을 가지고 있는 파이썬 파일을 의미

[Ex] 계산기 모듈 만들어 사용하기

  • Module
  • Excute

[Ex] 로또 번호 난수

  • Module
  • Excute


모듈(Module) 사용

  • import 키워드를 이용하여 module을 가져온다.
  • as 키워드를 이용하여 모듈 이름을 단축시킬 수 있다.
  • from ~ as 키워드를 통해 모듈에서 특정 부분만 가져올 수 있다.
    • 모듈명으로 접근하지 않고, 기능(함수)명을 이용해서 접근할 수 있다.

[Ex] 계산기

  • Module
  • Excute(as)
  • Excute(from ~ as)
  • .append() : 리스트에 추가




전역변수

전역변수 name

  • name에는 모듈 이름이 저장된다.
  • 실행파일인 경우, main 이 저장된다.
addModule.py name = ‘addModule’
subModule.py name = ‘subModule’
mulModule.py name = ‘mulModule’
divModule.py name = ‘divModule’
module.py name = ‘main’

실행 파일 지정

[Ex] cm를 mm, inch, m, ft로 변환해보자.

  • 직접 실행
  • 모듈(Module)로써 불러와 실행




패키지

패키지를 이용하면 관련 있는 모듈을 그룹으로 관리할 수 있다

[Ex] 계산기

모듈

  • 정수형/실수형 모듈 각각 작성
  • 정수형 모듈
  • 실수형 모듈
  • 실행 차이


site-packages

  • 실행파일과 동일한 디렉토리에 있는 경우만 모듈을 사용할 수 있음(다른 디렉토리에 있는 경우 모듈 사용 불가)
  • site-packages에 있는 모듈은 어디서나 사용할 수 있는 범용 모듈화 가능

sys : 시스템에 대한 간략한 정보를 모아놓은 Python 기본 제공 모듈(Module)
sys.path : 모듈이 들어있는 경로 값을 나열

  • venv : Python을 작업하기 위한 가상 환경. 독립적인 환경 셋팅을 할 수 있는 큰 장점
  • 독립 환경이 구성되지 않을 경우, 다른 버전 등(ver 3.7, ver 3.9)에 의해 에러(error)가 발생할 수 있음
