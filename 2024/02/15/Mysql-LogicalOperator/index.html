(MySQL) 논리 연산자

Created2024-02-15|Updated2024-02-15|StudyMySQL
|Post Views:

실습환경 만들기


Logical Operations

연산자 의미
AND 조건값을 모두 만족하는 경우 TRUE
OR 하나의 조건이라도 만족하는 경우 TRUE
NOT 조건을 만족하지 않는 경우 TRUE
BETWEEN 조건값이 범위 사이에 있으면 TRUE
IN 조건값이 목록에 있으면 TRUE
LIKE 조건값이 패턴에 맞으면 TRUE


AND

  • 조건을 모두 만족하는 경우 TRUE
    • 조건을 하나라도 만족하지 못하는 경우 검색대상에서 제외

[Ex] 나이가 29세이고 성별이 여성인 데이터 검색


Practice

[Ex] 성별이 남자이고 나이가 40세보다 큰 데이터를 이름의 역순으로 정렬하여 검색

[Ex] celeb 테이블에서 성별이 남자이고 소속사가 YG엔터테이먼트인 데이터를 소속사 순으로 정렬하여 조회

[Ex] celeb 테이블에서 나이가 30세 이상이고 성별이 남자인 데이터를 나이순으로 정렬하여 조회


OR

  • 하나의 조건이라도 만족하는 경우 TRUE

[Ex] 나이가 25세보다 작거나 30세보다 큰 데이터 검색

  • AND와 OR가 동시에 쓰일 경우, AND가 우선순위로 처리됨
    • 괄호 ()를 이용하여 조건을 묶어주는 것 중요!

[Ex] 나이가 29세보다 작고 여자이거나, 나이가 30세보다 크고 남자인 데이터를 나이와 성별 순으로 정렬

[Ex] YG엔터테이먼트 소속이거나 나무엑터스 소속인 연예인 중, 나이가 30보다 작은 데이터를 검색


Practice

[Ex] Celeb 테이블에서 소속사가 YG 엔터테이먼트이거나 안테나인 데이터를 소속사 순으로 정렬

[Ex] Celeb 테이블에서 남자이면서 YG엔터테이먼트 소속이거나, 나이가 30세보다 작은 데이터를 나이, 소속사 순으로 정렬하여 검색

[Ex] Celeb 테이블에서 아이디가 홀수면서 성별이 남자이거나, 아이디가 짝수면서 소속사가 YG엔터테이먼트인 데이터를 나이순으로 정렬


NOT

  • 조건을 만족하지 않을 경우 TRUE

[Ex] 성별이 여자가 아닌 데이터 검색

[Ex] 소속사가 YG엔터테이먼트이면서 남자가 아니거나 직업이 가수이면서 소속사가 YG엔터테이먼트가 아닌 데이터 검색

[Ex] 생일이 1990년 이후이면서 여자가 아니거나, 생일이 1979년 이전이면서 소속사가 안테나가 아닌 데이터 검색


Practice

[Ex] Celeb 테이블에서 소속사가 YG엔터테이먼트가 아니고 나이가 40세 이하인 데이터를 이름순으로 정렬하여 조회

[Ex] Celeb 테이블에서 성별이 남자가 아니거나 나이가 30세 이상인 데이터를 나이의 역순으로 정렬하여 조회

[Ex] Celeb 테이블에서 직업이 가수가 아니면서 성별이 여자이거나, 나이가 40보다 작지 않으면서 아이디가 홀수인 데이터를 조회


BETWEEN

  • 조건값이 범위 사이에 있으면 TRUE

[Ex] 나이가 20세에서 40세 사이의 데이터 검색

[Ex] 생년월일이 1980년에서 1995년 사이가 아니면서 여자이거나, 소속사가 YG엔터테이먼트이면서 나이가 20세에서 45세 사이가 아닌 데이터 검색


Practice

[Ex] Celeb 테이블에서 나이가 30세에서 60세 사이이고 성별이 남자인 데이터를 나이순으로 정렬하여 조회

[Ex] Celeb 테이블에서 나이가 30세에서 60세 사이가 아니거나 YG엔터테이먼트 소속인 데이터를 나이의 역순으로 정렬하여 조회

[Ex] Celeb 테이블에서 아이디가 1에서 5 사이의 값이면서 성별이 여자이거나, 아이디가 홀수이면서 성별이 남자이면서 나이가 20세에서 30세 사이인 데이터를 조회


IN

  • 목록 안에 조건이 존재하는 경우 TRUE

[Ex] 나이가 28세, 48세 중 하나인 데이터 검색

[Ex] 소속사가 나무엑터스, 안테나, 울림엔터테이먼트가 아니면서, 성별이 여자이거나 나이가 45세 이상인 데이터 검색


Practice

[Ex] celeb 테이블에서 아이유, 이미주, 유재석, 송강 중에 소속사가 ‘나무엑터스’인 데이터를 조회

[Ex] celeb 테이블에서 소속사가 안테나, YG엔터테이먼트 중 하나가 아니고, 성별이 여자인 데이터를 조회

[Ex] celeb 테이블에서 아이유, 송강, 강동원, 차승원 중에 YG엔터테이먼트 소속이 아니거나 나이가 40세에서 50세 사이인 사람 조회


LIKE

  • 조건값이 패턴에 맞으면 TRUE

[Ex] 소속사 이름이 ‘YG엔터테이먼트’인 데이터를 검색

[Ex] ‘YG’로 시작하는 소속사 이름을 가진 데이터 검색

[Ex] ‘엔터테이먼트’로 끝나는 소속사 이름을 가진 데이터 검색

[Ex] 직업명에 가수가 포함된 데이터를 검색

[Ex] 소속사 이름의 두 번째 글자가 G인 데이터를 검색

[Ex] 직업명이 ‘가’로 시작하고 최소 2글자 이상인 데이터 검색

[Ex] 직업명이 ‘가’로 시작하고 최소 5글자 이상인 데이터 검색

[Ex] 직업명이 ‘영’으로 시작하고 ‘모델’로 끝나는 데이터 검색

[Ex] 영화배우와 텔런트를 병행하는 연예인 검색

[Ex] 직업이 하나 이상인 연예인 중 영화배우 혹은 텔런트가 아닌 연예인 검색


Practice

[Ex] celeb 테이블에서 직업 중 가수가 포함되어있고 성이 이씨인 데이터 조회

[Ex] celeb 테이블에서 성별이 남자이거나 직업명이 ‘텔런트’로 끝나면서 최소 5글자 이상인 데이터를 조회

[Ex] celeb 테이블에서 이름이 두글자인 데이터를 조회

[Ex] celeb 테이블에서 나이가 30세 이상 50세 이하면서 개그맨이 아닌 데이터를 조회

[Ex] celeb 테이블에서 아이유, 이미주, 유재석, 송강 중에 소속사 이름이 ‘나무’로 시작하는 데이터를 조회

[Ex] celeb 테이블에서 아이유, 이미주, 송강, 이수현 중에 가수만 직업으로 가졌거나, 가수를 병행하지 않고 텔런트를 하는 사람을 찾으세요.
