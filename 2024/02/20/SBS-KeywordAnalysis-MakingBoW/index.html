(Step) 키워드, 시간대별 응답률 분석 - 자연어 처리 (KoNLPy)

KoNLPy

Local 환경에서 KoNLPy 패키지 사용이 불가하여 Google Colab을 통해 진행

  • Java 및 KoNLPy 동작에 필요한 패키지 설치
  • 패키지 로드 및 환경변수 설정
  • FigureOut 및 패키지 로드
  • 파일 경로 설정


Data Preprocessing

영문이 아닌 한글을 처리해야 하기 때문에 “KoNLPy”이라는 library를 사용


형태소 분석 - 명사단위

  • 명사 형태소 추출 함수 Okt()를 이용하여 정규표현식을 적용한 msg 내용의 형태소를 추출
  • 전체 말뭉치(corpus)에 적용해서 명사 형태소를 추출
  • 정규직 적용하여 추출
  • 전체 말뭉치(corpus)에서 명사 형태소 추출
  • 단어별 빈도(Frequency) 수 탐색
  • 한 글자 명사 제거
    • 두 글자 키워드 : 대부분 의미 있는 단어
    • ‘수’, ‘것’, ‘저’와 같은 한 글자 키워드는 분석에 딱히 좋은 영향을 미치지 않을 것으로 보임
    • 한 글자 명사를 제거
  • 불용어 사전을 통한 제거
    • 한 글자 명사 제거 후, ‘우리’, ‘매우’ 와 같은 실질적인 의미가 없고 꾸민 역할을 하는 불용어들이 아직 존재
    • 한국어 불용어 사전을 정의하여 불용어도 제거


Word Count

  • BoW(Bag of Words) 벡터 생성
    • BoW : 단어들의 출현 빈도(frequency)에만 집중하는 텍스트 데이터의 수치화 표현 방법
      • (1) 각 단어에 고유한 정수 인덱스를 부여 # 단어 집합 생성
      • (2) 각 인덱스의 위치에 단어 토큰의 등장 횟수를 기록한 벡터를 만듬
  • 단어 리스트
  • 단어별 등장 횟수
  • 단어 - 총 등장 횟수


TF-IDF 적용

  • Bag of Words 벡터에 대해서 TF-IDF변환 진행
    • 변환 후 matrix가 출력
    • 한 행(row)은 한 채팅을 의미
    • 한 열(column)은 한 단어를 의미
  • 첫 번째 리뷰에서의 단어 중요도(TF-IDF 값) — 0이 아닌 것만 출력
  • Vector - Word Mapping
