(Step) 키워드, 시간대별 응답률 분석 - EDA

In_and_Out 인원 파악

  • kakaotalk 채팅방 유입/이탈 인원을 확인하기 위해 데이터 셋 생성
    • action_msg 확인 결과 들어오기, 나가기, 메시지 가리기 3개의 항목으로만 구성되어 있음
    • 들어온 인원의 경우 0, 나간 인원의 경우 1, 메시지 가리기인 경우 2로 분류 진행


전체 유입/이탈 인원 파악 (일)

  • 시간 기준 : 일별
  • action_msg 중 ‘메시지 가리기’를 제외, 유입/이탈 인원을 count로 계산
    • Red Solid Line : Out
    • Blue Dotted Line : In
  • talk_date 데이터타입이 object로 확인하여 datetime으로 변경
    ※ 그래프를 시계열로 그려 확인하기 위해
  • 일별 시간을 기준으로 유입/이탈 인원 수를 입/퇴장 여부(action_msg)를 기준으로 시계열 그래프로 표현


월 단위 유입/이탈 인원 파악

  • 월별 전체 유입/이탈 인원 파악
    • 월(month)을 기준으로 전체 유입/이탈 인원 count
  • 월(month) 기준 유입/이탈 인원 수를 입/퇴장 여부(action_msg)를 기준으로 시계열 그래프로 표현
    • Red Solid Line : Out
    • Blue Dotted Line : In
    • 오픈채팅방에 들어오는 인원은 지속적으로 감소하는 추세를 보임
    • 12월 이상적으로 낮은 count를 확인


월(Month)에 따른 일(Day)별 유입/이탈 인원 파악

  • 가시성을 위해 월(Month) 숫자형을 영문 표기형으로 변경
  • 월(Month)에 따른 일(Day)별 유입/이탈 인원을 시계열 그래프를 표현
    • Red Solid Line : Out
    • Blue Dotted Line : In
    • 12월의 경우 데이터가 23-12-02까지 존재하여 count 수가 낮음

요일별 유입/이탈 인원 파악

  • 요일별 유입/이탈 인원 파악을 위해 데이터 셋 생성
    • 그래프를 그리기 이전 요일 정렬을 위해 custom_order 리스트 생성

  • 요일별 유입/이탈 인원을 파악하기 위해 시계열 그래프로 표현

    • Red Solid Line : Out
    • Blue Dotted Line : In

  • 평균적으로 월요일에 유입/이탈이 활발한 것을 확인

  • 주말(토, 일)요일의 경우 유입/이탈 인원이 확연히 감소


메세지 갯수를 통한 활동량 파악

활동량 기준 : 메세지의 갯수

  • 활동량 파악을 위한 데이터프레임 생성


월별 전체 활동량 파악

  • 8월, 12월의 경우 기간의 차이로 인해 적게 나타남
  • 전체적으로 비슷한 활동량을 보임


요일별 전체 활동량 파악

  • 요일별 활동량 파악
  • 요일별 활동량을 막대그래프로 표현
    • 주중(월, 화, 수, 목, 금)의 경우 활동이 활발하게 일어남
    • 주말(토, 일)의 경우 활동량이 미비함
  • 월별 요일별 활동량 파악
    • 전체적으로 주중에 공통적으로 활발하게 활동하는 것으로 나타남
    • 그러나, 특정 요일에 활발하다는 결론은 내지 못함

시간대별 활동량

  • 활동시간(09시 ~ 18시) 내 커뮤니케이션이 활발하게 일어남
  • 특히, 15 ~ 17시 가장 높은 분포를 보임


채팅 참여자 파악

채팅 참여 여부 판단

  • 채팅 여부 판단
    • 채팅방 내 채팅에 참여한 사람과 참여하지 않은 사람의 비율을 파악
    • 참여자의 경우 317명(65%), 미참여자의 경우 167명(35%)으로 나타남


요일별 채팅 참여 여부 파악

  • 요일별 채팅 참여자 수 파악
  • 요일별 채팅 참여자 및 대화내용 수를 막대그래프로 표현
  • 채팅 참여의 경우 주중에서도 화요일, 수요일에 특히 활동이 많은 것을 확인


직군별 참여자 파악

  • 같은 직군임에도 불구하고 직군의 이름이 다양하게 분류
    ex) 데이터분석, 데이터분석가, DA
  • 직군 분류 2차 가공 진행
  • 대학생, 데이터분석가, 마케터 직군 순으로 많은 참여자가 분포
2차 가공 참여자 분포


참여자 별 대화내용 수 파악

  • 채팅 참여자를 기준으로, 대화내용 수 파악을 위해 요일별 채팅 수 데이터프레임 생성
  • groupby를 통해 참여자별 전체 채팅 갯수 내림차순으로 표시
    • 이 중, 전체 채팅 수가 많은 20명의 참여자를 chat_top20에 할당
  • 채팅수가 많은 Top20의 채팅 수를 막대그래프로 표현


경력별 참여자 파악

  • 대학생의 경우, 0년차(신입)으로 가정
  • 경력이 높아질수록 대화자의 수가 적어짐
  • 전체 대화의 33%가 0년차(신입)에서 이뤄지며, 인원대비 대화 참여율은 10년차가 가장 높게 나타남


이탈자 채팅 여부 파악

  • 이탈자 파악을 위해 데이터프레임 생성
  • 이탈자 채팅 횟수를 파악
    • 이탈자 중 채팅에 참여했던 인원 파악 : 14명
  • 이탈자 중 채팅 참여자 수 및 비율을 그래프로 표현
    • 이탈자 중 채팅 참여비율은 전체 127명 중 14명(11%)로 확인
  • 이탈자 중 채팅 참여자의 채팅 수 확인
    • 가장 많이 채팅한 이탈자의 경우 7건으로 나타남


질문&답변 키워드 파악

키워드 빈도 수

  • 질문과 답변을 구분할 수 있는 키워드를 탐색하기 위해 채팅 내 키워드 빈도수를 확인
채팅 내 키워드 빈도 수 Top 10 질문자의 최초 질문 키워드 Top 10
  • ‘데이터’, ‘이벤트’, ‘프로젝트‘, ’페이지‘ 단어의 경우 전체 채팅과 질문자의 채팅에서 두드러지게 나타남
  • 질문과 답변을 구분하는 키워드로는 적절하지 않다고 판단


시간대별 답변 수

  • 활동시간(09:00 ~ 18:00) 내 일반적으로 답변이 활발하게 일어남
  • 10:00, 15:00, 09:00 순으로 가장 많은 답변을 보임
  • 채팅 활동량과 답변 수를 비교하였을 떄, 오전 10시, 오후 9시 이후 약 10%를 차지
