  • General Fomula
  • $\mu(X_t, t)$ is the drift term, which describes the deterministic part of the change.
  • $\sigma(X_t, t)$ is the diffusion term, which represents the contribution of the stochastic (random) process.
  • $dW_t$ is the differential of a Wiener process (Brownian motion), which represents the random fluctuations.

이해하기

  • 좌변 : 우리가 구하고 싶은 함수 $S(t)$의 움직임
  • 우변
    • 첫 항 $\mu(X_t, t)dt$ : 움직임이 예측가능한 부분(deterministic)
    • 둘째 항 $\sigma(X_t, t) dW_t$ : 움직임이 예측불가능한 부분(stochastic)


Standard Brownian Motion

  1. $W_0 = 0$
  2. (Independent Increments) For any , the random variable are independent.
  3. (Stationary) $W_t - W_s \sim N(0, t-s)$, for $0 \leq s < t$


Properties of Brownian Motion

  1. Markov property
  2. Martingale property
  3. Gaussian process
  4. $\frac{1}{2}$-self similar property
  5. 미분 불가능
  6. Brownian Motion의 sample path는 오직 $p>2$일 때, bounded $p$-variation을 갖는 유계 $p$-변동함수이다.
  7. Reflection Property

Bounded P-varaiation

함수 $f:[a, b] \rightarrow \mathbb{R}$가 적당한 $p>0$에 대해서 다음 조건을 만족할 때 $f$는 Bounded $p$-variation이라고 한다.

  • 여기서 supremum은 $[a, b]$의 모든 분할 $r_n : a = t_0 < t_1 < t_n = b$에 대해 취한다.
  • Brownian motion의 표본경로는 $p>2$일 때 폐구간 $[a,b]$에서 bounded $p$-variation이고 $0 < p \leq 2$인 경우 유계변동이 아니다.
    • 유한이면 리만스틸체스 적분 가능한데, 안 된다.
  • 따라서, 함수 $f$가 $[a, b]$에서 연속이라도, Riemann-Stieltjes 적분 $\int_{a}^{b}f(t)dW_t$가 존재하지 않는 경우가 발생
