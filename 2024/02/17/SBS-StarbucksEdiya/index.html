(Step) 스타벅스, 이디야 매장 데이터 분석

Created2024-02-17|Updated2024-02-20|StudyStep By Step
|Post Views:

Summary

  • Goal : 스타벅스, 이디야 매장 데이터 분석
  • 수행기간 : 2023. 10. 09 ~ 2023. 10. 16
  • 수행내용
    • 서울 내 스타벅스의 매장은 606개, 이디야는 636개로, 비슷한 양상을 보임
    • 구별로 보았을 때, 강남구가 가장 많았으며, 중구, 영등포구, 서초구 순으로 매장 갯수가 많음을 보임
    • 스타벅스와 이디야 매장을 비교하였을 때, 대체적으로 유사한 경우는 영등포구, 마포구, 송파구, 양천구, 광진구로 보여짐
    • 그 외 구에서는 특이점을 찾지 못함
      • 스타벅스 비중이 많은 구 : 강남구, 중구, 서초구, 종로구, 용산구
      • 이디야 비중이 많은 구 : 중랑구, 성북구, 구로구, 관악구, 은평구, 동대문구, 도봉구, 동작구, 강북구 등
    • 스타벅스(Red)와 이디야(Blue) 매장을 지도에 표시하였을 때, ‘이디야 커피는 스타벅스 매장 근처에 있다’ 라고 할 수는 없다고 보여짐


스타벅스 매장 분석

  • Module Load


Chrome Driver 설정

  • 스타벅스 매장 찾기 홈페이지 접속하여 화면 최대화 설정
  • 서울 매장검색을 위해 [지역] - [서울] - [전체] 클릭


데이터 불러오기

  • Beautifulsoup 통해 데이터 불러오기
  • 매장별 정보 가져오기
  • 매장 정보를 추출하기 위해 임의의 행을 선택하여 확인

Output : '서울특별시 강남구 삼성로 402 (대치동)1522-3232’

Output : '대치사거리 ‘

Output : '강남구’

  • 수집한 정보를 가져와 이름, 주소, 구 이름에 해당하는 데이터 리스트 생성
  • 데이터 프레임 생성
  • 부산 지역 4개 지점이 같이 수집되어, 부산 지역 매장 삭제

데이터 저장 (스타벅스)


EDIYA 매장

  • 이디야 매장 찾기 홈페이지 접속하여 화면 최대화 설정
  • 스타벅스 매장 위치가 있는 ‘구’ 호출
  • 홈페이지에서 구를 입력 후 검색
  • 현재 창 주소를 저장
  • 매장별 정보를 추출하기 위한 부모 태그

Output : <li class="item"><a href="#c" onclick="panLatTo('127.02810578707652','37.51408005446769','2');fnMove();"><div class="store_thum"><img src="../images/customer/store_thum.gif"/></div><dl><dt>강남논현학동점</dt> <dd>서울 강남구 논현로131길 28 (논현동)</dd></dl></a></li>

  • 필요 정보 추출을 위해 임의 행 지정하여 구, 매장이름, 주소 확인

Output : '강남구’

Output : '강남논현학동점’

Output : '서울 강남구 논현로131길 28 (논현동)’

  • 해장 지역(구)에 대한 정보 수집 후, 검색창 지우기


Error

  • 실행코드를 작성하여 돌렸으나, 오류가 발생
    • 원인 : 중구의 경우, 매장 수가 많아 검색 결과 오류 발생
  • 따라서 ‘중구’ column명을 ‘서울 중구’로 변경
  • 정보 수집 및 각 정보(구 이름, 매장이름, 주소)를 리스트에 저장

데이터 프레임 생성 및 저장


EDIYA/스타벅스 매장 간 관계 분석

  • 데이터 불러오기
  • 스타벅스, 이디야 구분을 위해 brand column 추가
  • 하나의 데이터 프레임으로 변환


전체 매장 갯수

서울시 구별 총 매장 갯수

  • 서울시 구별 각 브랜드 매장 갯수


매장 위치 지도 시각화

  • Googlemaps를 활용하여 각 매장의 위경도 정보 가져오기
  • Googlemaps를 활용하여 각 매장의 위경도 정보 가져오기
  • 위경도 정보를 병합
    • 주소를 가져오지 못하는 매장의 경우, drop 처리 진행
