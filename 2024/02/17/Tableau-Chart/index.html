차트 만들기

마크 - 사각형 변경


막대 차트

  • 특정 집계에 대한 절대적 크기를 시각적으로 표현하기 적합
  • 집계에 대한 단순한 크기를 보여주고자 할 때 사용


누적 막대그래프

  • 고객 세그먼트를 효과적으로 보기
  • 전체 매출의 비율로 계산
  • 연별 비중을 알고 싶을 경우 테이블(아래로) 클릭


라인 차트

  • 1년의 값에 대한 시간별 추세 확인
  • 미래값을 예측할 경우 유용
  • 분석을 위해 패널 라인 추가
  • 평균값을 확인하기 위해 서식에서 지정


파이 차트

  • 원 안에서 전체를 기준으로 한 부분의 상대적 크기를 나타냄
  • 비중을 표기할 때 유용
  • 비중을 제대로 나타기 위해서는 [각도] 이용


트리맵 차트

  • 전체를 기준으로 차지하는 비중을 보여주는 차트
  • 데이터 항목이 많은 경우 파이차트보다 트리맵 차트 사용이 더 효과적
  • 트리맵 선택


분산형 차트

  • 측정값 관계를 파악하기 위해 사용
  • 2~4개 측정값을 올린 뒤, 차원으로 나누는 방식으로 세분화하여 정보를 확인
  • 추세선과 같은 정보를 추가하여 두 측정값 간의 관계를 파악
  • 기준값 0 확인
  • 추세선 표현


  • 데이터 설명


이중축 차트

  • 하나의 뷰 안에서 축을 이중으로 써서 차트를 만드는 경우
  • 제한된 공간에서 여러 값을 비교하기 위해 사용
  • 마크를 서로 동일하게 사용 가능하며, 다른 마크를 이용해 표현도 가능


콤비네이션 차트

  • 막대차트 + 라인차트
  • 더 많은 정보를 시각화할 때 사용
  • 목표치 대비 실제 매출액을 나타낼 때 주로 사용
  • 라인 차트
    • 트랜드와 함께 매출을 강조하는 데 사용
