(Tableau) 다양한 기능 활용

Created2024-02-17|Updated2024-02-17|ToolTableau
|Post Views:

그룹

  • 항목 내 원하는 만큼 자유롭게 그룹 가능
  • 눈금선 이동 (0.5)


집합

  • 집합에 포함되는지 아닌지로 결정
  • IN : 선택 그룹 (서울, 인천, 경기)
  • OUT : 기타

집합과 그룹의 차이

  • 두 개의 군을 비교할 경우 집합을 사용
  • 두 개 이상의 군을 비교할 경우 그룹을 사용

계층

  • 각 고객별 매출 상위 11위 ~ 20위 고객만 포함하는 집합을 만듬
  • 상위 10명 집합 생성
  • 상위 20명 집합 생성
  • 결합된 집합 만들기
    • 상위 11 ~ 20위를 만들기 위해 사용
  • 필터를 통해 Top 11 ~ 20위를 가져옴


계층이란?

  • 화면에서 드릴다운 하여 값을 세부적으로 찾는 데 유용
  • 지역데이터를 기준으로 동일한 시/군/구 명이 있을 경우 여러 에러가 발생할 수 있음
  • 이러한 경우를 대비하여 상위 레벨, 하위 레벨에 대한 계층을 만들어주는 것이 좋음


계층 만들기

  • 하위 개념인 차원 필드를 드래그해서 계층을 만들고자 하는 상위 필드에 올리기
  • 계층을 만들고자 하는 차원 필드를 선택 후 마우스 우클릭
  • 날짜 테이블의 경우 자동으로 계층이 형성
    • 년, 분기, 월, 일 형태로 계층이 형성
