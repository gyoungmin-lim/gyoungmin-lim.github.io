(Python) Package - Selenium

Created2024-02-17|Updated2024-02-17|ToolPython
|Post Views:

Selenium

링크 : https://www.selenium.dev/documentation/

주요 기능

  • 웹 브라우저를 원격 조작하는 도구
  • 자동으로 URL을 열고 클릭 등이 가능
  • 스크롤, 문자의 입력, 화면 캡쳐 등

Beautiful Soup만으로 해결할 수 없는 것

  • 접근할 웹 주소를 알 수 없을 때
  • 자바스크립트를 사용하는 웹 페이지의 경우
  • 웹 브라우저로 접근하지 않으면 안될 때


설치

  • 윈도우, Mac(intel)
  • Mac(m1)


Selenium 세팅

Chrome 버전 확인

  • 크롬 오른쪽 상단 [맞춤 설정 및 제어] - [도움말] - [Chrome 정보]
  • Chrome 정보 확인
  • Google에서 Chromedriver 검색
    • 링크 : https://chromedriver.chromium.org/downloads
    • 크롬 버전 앞 부분 숫자에 맞는 것 중 선택
      • 사용중인 OS 버전에 맞춰 다운로드


Selenium webdriver 사용하기

  • Module Load 및 크롬드라이버 경로 지정
    ※ Selenium Update로 인해 webdriver.Chrome() 크롬드라이버 경로지정 필요없음 (설치도 X)
  • 리소스 낭비하지 않게 driver.quit() 명령어 사용

  • 새로운 크롬이 나타나면서 지정된 웹 주소에 접근

  • 현재 브라우저 창 크기 확인


화면 스크롤

  • 스크롤 가능한 높이(길이) 가져오기
  • 화면 스크롤 : 하단 이동
  • 화면 스크린샷
  • 화면 스크롤 : 상단 이동

Selenium 태그 명령어

  • find_element_by_css_selector => find, select_one
  • find_elements_by_css_selector => find_all, select
  • 특정 태그 지점까지 스크롤 이동


검색어 입력

  • 입력 창에 글자 넣기
    ※ 현재 화면에 보이지 않으면 입력 X
  • 브라우저 창 크기 조절
    ※ 현재 보이는 화면에서만 가능
  • 화면 최대화
  • 화면 최소화
  • 새로 입력 시, 뒤에 추가로 입력
  • 초기화 후 검색어 입력


Xpath

  • Xpath 태그 찾는 방법
방법 내용
// 최상위
* 자손 태그 검색 => div form
/ 자식 태그 검색 => div > form
'td[2]' td 중에서 2번째 태그
  • Xpath 이용
    ※ BeautifulSoup에서는 Xpath 사용 불가능, Selenium에서는 사용 가능
  • css_selector 이용
  • 현재 화면의 html 코드 가져오기
