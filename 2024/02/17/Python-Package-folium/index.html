(Python) Package - Folium

Created2024-02-17|Updated2024-02-17|ToolPython
|Post Views:

Folium

설치


사용방법


Save Path

  • HTML 파일로 저장 가능


Tile option

  • tile : 지도 스타일 지정
  • zoom_start : 시작배율 설정


folium.Marker()

  • 지도에 마커 생성


folium.ClickForMarker()

  • 지도 위에 마우스로 클릭했을 때 마커를 생성


folium.LatLngPopup()

  • 지도를 마우스로 클릭했을 때 위도/경도 정보를 반환


folium.Circle(), folium.CircleMarker()

  • folium.Circle() : 좌표에 원을 이용해 표시
  • folium.CircleMarker() : 좌표에 원을 이용해 표시

※ 둘 간의 차이점

  1. folium.CircleMarker()는 배율에 따라 Circle의 radius가 자동 조정 됨
  2. 그 외 옵션은 모두 동일


다양한 옵션들

folium.Circle() / folium.CircleMarker()


folium.Choropleth

  • 주어진 데이터
  • fill_opacity : 채우기 투명도
  • line_opacity : 선 투명도
  • legend_name : 범례 이름
