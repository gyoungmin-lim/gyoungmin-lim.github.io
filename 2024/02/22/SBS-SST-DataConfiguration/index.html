데이터 구성

(작성 필요)

  • 파일

데이터 불러오기

  • Module Load
  • Figure 저장소 설정
  • NetCDF File 열기
  • File 정보의 Key 확인
    • 위/경도, 분석오차, 시간, 빙하, 수온, 마스킹으로 구성

dict_keys(['lon', 'analysis_error', 'time', 'sea_ice_fraction', 'lat', 'analysed_sst', 'mask'])

  • 필요 정보를 변수에 저장
  • 파일 닫기

Visualization

  • 위/경도에 따른 수온을 시각화

수온의 상태가 이상하다

  • OSTIA NetCDF 수온의 경우, 켈빈(Kelvin, K)으로 단위가 지정되어 있음
  • 따라서, 켈빈을 섭씨($ ^oC$)로 변경해줄 필요가 있음
  • 수온 단위 변환 및 시각화
