(Step 2) LSTM 기법을 활용한 수온 예측 - EDA

Created2024-03-09|Updated2024-04-02|StudyStep By Step
|Post Views:

참고

  • (Step) 위성자료를 활용한 표층수온 예측 - Data Configuration


Goal

  • 데이터의 형태가 어떻게 나타나는지, 시간에 따른 표층수온(Sea Surface Temperature, SST)가 어떻게 나타나는지 확인


데이터 가공

이전에 진행하였던 데이터 구성에서 확인한 바와 같이, 데이터를 변한화는 과정은 동일하게 적용(참고문서 : (Step) 위성자료를 활용한 표층수온 예측 - Data Configuration)

  • Module Load 및 Figure Storage 생성
  • NetCDF 파일열기 및 필요변수 할당


Convert Time

  • Epoch Time을 현재 시간(Local Time)으로 변환
  • 변환된 시간을 ‘yyyy-mm-dd’와 ‘hh:mm’으로 변환


Kelvin to Cellcius

  • 켈빈(K)를 섭씨(C)로 변환
  • 변환된 데이터 shape 확인
    • 데이터 행의 경우, days(일 단위)로 되어있음을 알 수 있다.


Set Min, Max Temperature

  • 수온의 변화를 눈에 잘 보이게 하기 위해, 표층수온의 최대, 최소값을 구하고, 이를 Colorbar에 고정
    • 가지고 있는 데이터의 최대, 최소값으로 설정
    • Max : 31.25, Min : -2.0


Visualization

  • Cellicus 단위로 변환한 데이터를 시각화
  • Colorbar의 경우, 표층수온 데이터의 ‘최대, 최소’값으로 고정
2023-03-01 2023-06-01 2023-09-01 2023-12-01


Making GIF

  • 변화를 좀 더 시각적으로 보기 위해, GIF 애니메이션으로 표현
    • imageio 패키지 사용
  • 저장된 Figure png File을 리스트 지정 및 애니메이션 생성
