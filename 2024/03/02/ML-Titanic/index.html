데이터 탐색적 분석 (EDA)

  • Package Install
  • Load Data


생존상황

  • ax = subplots의 위치를 의미


성별에 따른 생존현황


경제력 대비 생존률

  • 경제력 대비 생존상황
    • 1등실의 생존 가능성이 높음
    • 여성의 생존률이 높음
    • 그렇다면, 1등실에는 여성이 많이 타고 있었나?


선실 등급별 성별 상황

  • 3등실에 20대 남성이 많은 것을 확인


나이별 승객 현황

  • 아이들과 20~30대가 많이 탑승한 것을 확인 가능


선실 등급별 연령별 생존률

  • 선실 등급이 높을수록 생존률이 높은 것을 확인


나이별 Grouping

  • pd.cut : 그룹핑(Grouping)하는데 사용되는 명령어
    • bins 구분짓는 기준
    • labels : 구분기준(bins)에 대한 이름
  • 나이, 성별, 객실 등급별 생존자 수 파악
    • 어리고, 여성이고, 1등실일수록 생존하기 유리했을까?


남/여 나이별 생존 상황

  • 여성의 경우 생존이 상당히 많음
  • 남성의 경우
    • 20대 ~ 40대까지 대부분 사망
    • 어린아이의 생존율이 높음


탑승객 신분 확인

  • 탑승객의 이름에서 신분을 확인할 수 있음
  • 정규식을 이용해 문장 사이 신분정보 추출
  • 신분을 나타내는 column 추가
  • 성별별로 본 신분
  • 사회적 신분을 정리
    • 신분 종류를 확인 후, 여성형 귀족과 남성형 귀족이름을 정리
  • 사회적 신분에 따른 생존율 평균
    • 평민 남성의 경우 생존율이 가장 낮음
    • 귀족 남성의 경우 평민 여성보다 생존율이 낮음
    • 귀족이라고 생존율이 높은 것은 아님을 알 수 있음




머신러닝을 활용한 생존자 예측

  • 데이터 구조 확인
  • 머신러닝을 위해 sex column을 숫자로 변경
  • sex column을 숫자로 라벨링
  • 결측치(Missing Data)가 있을 경우 drop


상관관계 분석


ML

  • 특성을 선택하고, 데이터를 나누기
    • 특성은 ['pclass', 'age', 'sibsp', 'parch', 'fare', 'gender']를 선정


Decision Tree
