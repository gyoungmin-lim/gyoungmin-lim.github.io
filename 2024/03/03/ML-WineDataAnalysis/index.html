(ML) 와인(Wine) 데이터 분석

Created2024-03-03|Updated2024-03-03|StudyML/DL
|Post Views:

Wine

  • 분류 문제에서 많이 사용하는 Iris 꽃 데이터만큼 알려지진 않았지만, 와인데이터도 많이 사용한다.
  • 인류 역사에서 최초의 술로 알려져 있다


와인 맛 분류


와인 데이터

  • 레드와인 품질 데이터

    • 링크 : https://archive.ics.uci.edu/ml/machine-learning-databases/wine-quality/winequality-red.csv

  • 화이트와인 품질 데이터

    • 링크 : https://archive.ics.uci.edu/ml/machine-learning-databases/wine-quality/winequality-white.csv


데이터 탐색

  • Data Load



  • Column의 종류
  • Merge Data
    • 레드와인을 1, 화이트와인을 0으로 구분하여 데이터프레임 합치기
  • wine quality
  • 와인별 quality
    • 레드와인, 화이트와인 대체적으로 5~6등급에 많은 숫자가 포진


레드 와인 / 화이트 와인 분류

  • 라벨 분리
  • Train / Test Set 분리
  • 훈련용과 테스트용이 레드/화이트와인에 따라 어느정도 구분되었는지 확인


Decision Tree 훈련

  • 학습 결과


데이터 전처리

Min-Max Scaler & Standard Scaler

  • 와인 데이터 몇 개 황목을 boxplot
    • 컬럼들의 최대/최소 범위가 각각 다르고, 평균과 분산이 각각 다르다.
    • 특성(feature)의 편향 문제는 최적의 모델을 찾는데 방해가 될 수도 있다.
  • MinMaxScaler, StandardScaler
    • 일반적으로 column 간의 범위의 격차가 심할 경우, 제대로 학습이 안될 수도 있다.
    • scaler를 적용하는 것은 효과적일 수도 있으나, 어떤 scaler가 좋은지는 해봐야 안다
      (Decision Tree는 효과가 없다.)
  • MinMaxScaler boxplot
  • StandardScaler boxplot


DecisionTree using Scaler

  • MinMaxScaler를 사용해 DT 훈련
  • StandardScaler를 사용해 DT 훈련


레드와인과 화이트와인을 구분하는 주요 특성

  • plot tree
  • 레드와인과 화이트와인을 구분하는 주요 특성
    • MaxDepth를 높일 경우 수치에도 변화가 옴


와인 맛에 대한 이진 분류

  • quality column을 이진화
    • 와인 quality 등급이 5 이상이면 1, (5 미만일 경우) 0으로 이진화
  • 레드/화이트 와인 분류와 동일 과정
  • 발생 이유를 알아보기 위해 treemap 확인
    • quality column으로 taste 컬럼을 만들었기 때문에 quality column 제거 필요
  • quality column 제거 후 다시 실행
  • 어떤 와인을 맛있다고 할 수 있을까?
    • class_name으로 SoSo와 Good을 선정
    • alcohol 도수를 기준으로 10.525 이상이면 Good이라고 판정
    • alcohol 도수가 10.525 이하이고 volatile acidity가 0.237 이하일 경우 Good으로 판정
